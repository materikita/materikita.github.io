30. Tolong bantu jawab dong dengan caranya! Jika [tex]65^{0} + 65^{1} + 65^{2} + 65^{3} + ... + 65^{2017}[/tex] dibagi oleh 8, maka sisanya adalah....


notasi [tex]a = b\mod n[/tex] menunjukkan bahwa a adalah sisa pembagian b oleh n

menggunakan sifat

[tex](a+b)\mod n =(a\mod n + b\mod n)\mod n[/tex]

bisa didapat bahwa

[tex](65^0 + 65^1 + 65^2 + \dots + 65^{2017}) \mod 8\\= ([65^0 \mod 8]+[65^1 \mod 8]+ \dots + [65^{2017} \mod 8]) \mod 8\\[/tex]

jika kita coba meninjau

[tex]65^0\mod8=1\\65^1\mod8=1\\65^2\mod8=1\\\dots[/tex]

kita tahu untuk semua n bulat positif [tex]65^n \mod 8 = 1[/tex]

kalau butuh pembuktian, kalau tidak bisa langsung menggunakan pola yang sudah ditemukan tadi

[tex]65^n \mod 8 = (64+1)^n \mod 8\\= \left(\begin{array}{ccc}n\\0\end{array}\right) 64^n1^0 +  \left(\begin{array}{ccc}n\\1\end{array}\right) 64^{n-1}1^1 + \left(\begin{array}{ccc}n\\2\end{array}\right) 64^{n-2}1^2 + \dots+ \left(\begin{array}{ccc}n\\n\end{array}\right) 64^01^n \mod 8\\=\left( 64(\dots) + \left(\begin{array}{ccc}n\\n\end{array}\right) 64^01^n\right) \mod 8\\= \left(64(\dots) +1\cdot1\cdot1\right)\mod 8\\= \left(64(\dots) +1)\right)\mod 8[/tex]

[tex]= (64(\dots)\mod8 +1\mod8))\mod8\\= \left(0 +1)\mod 8\\= 1\mod 8\\= 1[/tex]

bisa langsung dipakai

[tex](65^0 + 65^1 + 65^2 + \dots + 65^{2017}) \mod 8\\= ([65^0 \mod 8]+[65^1 \mod 8]+ \dots + [65^{2017} \mod 8]) \mod 8\\= (1+1+1+\dots+1)\mod8\\= 2018\mod 8\\\boxed{= 2}[/tex]
