14. Rosyid bersama teman-temannya pergi tamasya ke semarang .mereka berangkat dari jakarta pukul 05.30 .mereka berhenti di masjid untuk istirahat dan melakukan sholat.Rosyid dan teman-temannya melakukan sholat zuhur ,kemudian mengerjakan sholat asar .sholat yg di lakukan oleh Rosyid adalah sholat...A.jama' taqdimB.jama' Ta'khirC.QasarD.Wajib


Shalat yang dilakukan oleh rosyid adalah pelaksanaan shalat secara jamak ta'khir. Karena rosyid mengerjakan shalat dzuhur dan ashar pada jam 5.30 ( sore hari). Sehingga jawaban yang tepat adalah B

Pembahasan

Shalat jamak ta`khir adalah pengerjaan dua shalat wajib secara jamak yang dilakukan pada waktu shalat yang kedua. Pada contoh shalat shalat yang dilakukan oleh rosyid misalnya ia mengerjakan shalat dzuhur dan shalat ashar pada waktu shalat ashar yaitu pada sore hari tepatnya jam 5.30

Shalat wajib lima waktu  yang dapat kerjakan secara jamak ada 4 shalat yaitu shalat dzuhur dengan shalat ashar dan shalat maghrib dengan shalat isya. Adapun shalat subuh tidak dapat dikerjakan secara jamak. Jadi shalat subuh walaupun bagi seorang mushafir juga harus mengerjakan shalat subuh seperti biasa.

15. Kalau jadwal terbang dari Jakarta 05.05 waktu tiba ke Jayapura jam berapa ka kakak


Perlu diketahui bahwa zona waktu di Indonesia dibagi menjadi 3, yaitu Waktu Indonesia Barat, Tengah, dan Timur. WIB memiliki perbedaan waktu 1 jam dengan WITA, dan 2 jam dengan WIT.

Wilayah Jakarta termasuk kedalam zona WIB, sedangkan Jayapura termasuk kedalam zona WIT.

Jadi, bisa disimpulkan bahwa apabila di Jakarta 05.05, maka di Jayapura adalah 07.05.

Nah, untuk waktu tiba di Jayapura tinggal ditambah 2 jam dari waktu di Jakarta saja.

21. buat jadwal waktu sholat dengan program C++ perulangan dan percabangan menggunakan DEVC++


#include <iostream>

#include <ctime>

using namespace std;

int main() {

   // Mendapatkan waktu saat ini

   time_t now = time(0);

   tm *ltm = localtime(&now);

   int currentHour = ltm->tm_hour;

   int currentMinute = ltm->tm_min;

   // Menghitung waktu sholat berdasarkan perbedaan jam dengan waktu terkini

   int fajrHour = 5;

   int fajrMinute = 0;

   int dhuhrHour = 12;

   int dhuhrMinute = 30;

   int asrHour = 15;

   int asrMinute = 0;

   int maghribHour = 18;

   int maghribMinute = 0;

   int ishaHour = 20;

   int ishaMinute = 0;

   // Menampilkan jadwal waktu sholat

   cout << "Jadwal Waktu Sholat" << endl;

   cout << "-------------------" << endl;

   cout << "Waktu saat ini: " << currentHour << ":" << currentMinute << endl;

   if (currentHour < fajrHour || (currentHour == fajrHour && currentMinute < fajrMinute)) {

       cout << "Waktu Sholat Fajr: " << fajrHour << ":" << fajrMinute << endl;

   }

   if (currentHour < dhuhrHour || (currentHour == dhuhrHour && currentMinute < dhuhrMinute)) {

       cout << "Waktu Sholat Dhuhr: " << dhuhrHour << ":" << dhuhrMinute << endl;

   }

   if (currentHour < asrHour || (currentHour == asrHour && currentMinute < asrMinute)) {

       cout << "Waktu Sholat Asr: " << asrHour << ":" << asrMinute << endl;

   }

   if (currentHour < maghribHour || (currentHour == maghribHour && currentMinute < maghribMinute)) {

       cout << "Waktu Sholat Maghrib: " << maghribHour << ":" << maghribMinute << endl;

   }

   if (currentHour < ishaHour || (currentHour == ishaHour && currentMinute < ishaMinute)) {

       cout << "Waktu Sholat Isha: " << ishaHour << ":" << ishaMinute << endl;

   }

   return 0;

}

28. Aziz berjalan ke Bandung mau ikutan demo di senayan Jakarta bersama teman- temannya. ketika berangkat pas waktunya dhuhur, apakah aziz harus melakukan sholat jamak taqdim dan di qasar sholatnya di rumah?Jawab yang bener ya ​


Azis tidak diperbolehkan melakukan Sholat Jamak taqdim dan meng qasar sholatnya di rumah, akan tetapi Azis boleh melakukan sholatnya di jamak apabila masih didalam waktu perjalannya. karena tidak memungkinkan perjalanan yang ditempuh dari bandung menuju ke jakarta di Qasar pada waktu kembali lagi pulang ke Bandung

Penjelasan:

Sholat Jamak taqdim  merupakan penggabungan dua shalat wajib yang dikerjakan pada waktu yang pertama (awal).  Jama' taqdim sendiri terdiri dari dua macam :

Menggabungkan shalat zuhur dan shalat ashar, dikerjakan pada waktu shalat dzuhur. Menggabungkan shalat magrib dan isya, dikerjakan pada waktu shalat magrib.

Adapun dalil pokok dari para ulama sebagaimana firman Allah ﷻ dalam Al-Qur’an, yaitu:  

وَإِذَا ضَرَبْتُمْ فِي الْأَرْضِ فَلَيْسَ عَلَيْكُمْ جُنَاحٌ أَنْ تَقْصُرُوا مِنَ الصَّلَاةِ  

Artinya, “Dan apabila kamu bepergian di bumi, maka tidaklah berdosa kamu meng-qashar shalat” (QS An-Nisa’: 101).

Shalat Jamak Qasar mempunyai arti memperpendek/meringkas. Dalam istilah ilmu fiqih, yaitu memperpendek rakaat shalat wajib, dari 4 rakaat menjadi 2 rakaat, sebagai dispensasi (rukhsah) bagi para musafir. Hanya saja, tidak semua musafir dan tidak setiap perjalanan mendapatkan dispensasi qashar.

Ada beberapa syarat yang harus dipenuhi terlebih dahulu menurut Habib Hasan al-Kaf untuk mendapatkan keringanan ini, diantaranya :

Bertujuan pada tempat yang telah ditentukan. Perjalanan bersifat mubah, bukan karena kemaksiatan. Perjalanannya karena tujuan yang baik, seperti berdagang, haji, silaturahim, dan sebagainya. Perjalanannya mencapai 2 marhalah. Sedangkan ukuran yang berlaku untuk diterapkan saat ini, yaitu, kurang lebih 89 km (88,704 km). Telah melewati batas desa. Masih ada dalam perjalanan sampai shalat selesai. Mengetahui hukum diperbolehkannya meng-qashar shalat. Shalat yang diqashar adalah shalat 4 rakaat.
