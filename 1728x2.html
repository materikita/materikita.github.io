15. Diketahui f(x) = 2x2 - 6x, g(x) = 2x + 3,dan h(x) = 4x. Fungsi (fog h)(x) = ....A. 128x^2 + 48x + 36B. 128×^2 + 48xC. 128x2 + 48x - 36D. 128x2 - 48xE. 128x2 - 48x - 36​


Jawaban:

Nilai dari (fogoh)(x) adalah 128x² + 48x

Jawaban : B

Pembahasan :

(fog)(x) = f(g(x))

(gof)(x) = g(f(x))

(g ± f)(x) = g(x) ± f(x)

(g • f)(x) = g(x) • f(x)

(fogoh)(x) = f(g(h(x)))

Penjelasan :

Diketahui f(x) = 2x² - 6x , g(x) = 2x + 3 dan h(x) = 4x

Maka, (fogoh)(x) adalah..

= (fogoh)(x)

= (goh)(x)

= g(h(x))

= 2(4x) + 3

= 8x + 3

= f . (g(h(x))

= 2(8x + 3)² - 6(8x + 3)

= 2(64x² + 48x + 9) - 48x - 18

= 128x² + 96x + 18 - 48x - 18

= 128x² + 48x

Kesimpulan :

Jadi, nilai dari (fogoh)(x) adalah 128x² + 48x
