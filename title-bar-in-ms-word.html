30. cara membuat title heading pada MS Word adalah..​


Jawaban:

Cara Membuat Heading di Word dengan Mudah

Membuat heading di Microsoft Word sangatlah mudah. Kalian hanya perlu mengikuti tutorial cara membuat heading di word berikut:

Buka aplikasi Microsoft Word (double click pada ikon Word di Desktop)

Lalu pilih new document atau blank document

Selanjutnya tuliskan teks “Heading 1” untuk dijadikan judul (H1)

Di bagian atas (menu bar), tekan Home -> Styles -> H1 (contoh ada pada gambar di bawah).

Selanjutnya tekan enter, kemudian tuliskan teks “Heading 2” untuk dijadikan sub judul (H2).

Tekan Home -> Styles -> H2.

Selesai.

33. mengenal lembar kerja word title bar menu bar toolbar


-Menu bar adalah daftar menu yang tersedia untuk program tertentu.
-Jenis jenis menu bar :
-Menu file berfungsi untuk mengatur file dokumen seperti menyimpan, membuka, mengatur kertas dokumen, dsb.
-Menu Edit berfungsi untuk melakukan pengeditan dokumen
-Menu View berfungsi untuk mengatur bagaimana tampilan dokumen
-Menu Insert berfungsi untuk menambahkan sesuatu ke dalam dokumen.
-Menu Format berfungsi untuk mengatur format sebuah dokumen. Seperti mengatur format huruf, paragraf dll
-Menu Tools mempunyai fungsi untuk mengatur program kerja
-Menu Table berfungsi untuk membuat tabel dan mengatur tabel tersebut
-Menu Windows berfungsi untuk mengatur lembar kerja
-Menu help berfungsi untuk memberi bantuan kepada kita yang belum tahu

-Toolbar Berisikan ikon untuk mem-format obyek-obyek pada dokumen. Obyek tersebut bisa berupa gambar, tabel, atau text dan sebagainya. Toolbar bisa juga digunakan untuk memilih berbagai jenis huruf dan ukurannya
