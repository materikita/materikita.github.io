30. Soal Matematika 1.) Uang Yusuf : Uang Aisyah = 3 : 4 uang Yusuf = 225.000 Ditanya : Berapa Uang Aisyah?2.) Uang Dian : Uang Septi = 4 : 7 .Selisih Uang Mereka 15.000 Berapa Uang Dian Dan Septia?​


Pembahasan

1. Uang Yusuf : Uang Aisyah = 3 : 4

Uang yusuf = 225.000

Uang Aisyah

= 4 / 3 x 225.000

= 4 x 75.000

= 300.000

2. Uang dian : Uang Septi = 4 : 7

Selisih = 15.000

Uang dian

= 4 / ( 7 - 4 ) x 15.000

= 4 / 3 x 15.000

= 4 x 5.000

= 20.000

Uang Septi

= 7 / ( 7 - 4 ) x 15.000

= 7 / 3 x 15.000

= 7 x 5.000

= 35.000
