1. Design ui yang dipakai oleh aplikasi disimpan dalam file dengan ekstensi A. nib B. car C. march 0 D. plist


Jawaban:

D. Plist

Penjelasan:

File dengan ekstensi .plist (Property List) digunakan untuk menyimpan data dalam format XML atau Binary. Format ini umumnya digunakan untuk menyimpan pengaturan, preferensi, konfigurasi, atau data lain yang digunakan oleh aplikasi. Plist file biasanya digunakan di lingkungan iOS dan macOS untuk menyimpan desain UI, pengaturan aplikasi, dan data terstruktur lainnya.

Maaf kalau salah and have a nice day!

Penjelasan:

Design UI yang dipakai oleh aplikasi disimpan dalam file dengan ekstensi:

A. nib (NeXTSTEP Interface Builder) - Jawaban ini mengacu pada file-interface yang digunakan dalam pengembangan aplikasi iOS atau macOS dengan menggunakan bahasa pemrograman Objective-C atau Swift.

File nib adalah file biner yang berisi informasi tentang tata letak UI, kontrol, dan properti UI lainnya yang digunakan dalam aplikasi. Ini adalah format file yang lebih lama dan telah digantikan oleh file storyboard dalam pengembangan aplikasi terbaru.

Jadi, jawaban yang benar adalah A. nib.

Contoh dari ekstensi file yang digunakan untuk menyimpan data yaitu:

Doc merupakan salah satu ekstensi file yang memungkinkan pengguna menyimpan file berisi teks, gambar, table, dan berbagai komponen lainnya.TXT adalah suatu bentuk jenis ekstensi file dokumen berisi teks mentah yang dapat dibuat dengan semua program pengolah kataHTML yaitu ekstensi file ini merupakan format penyimpanan data yang terhubung dengan halaman website.XLS yaitu jenis dari ekstensi file untuk menyimpan file spreadsheet seperti yang biasa digunakan dalam aplikasi Microsoft Excel.Pembahasan

Ekstensi file merupakan suatu bentuk penanda yang menunjukkan isi suatu berkas (file). Ekstensi file dalam bidang komputer biasanya akan digunakan oleh sistem operasi untuk menentukan program/aplikasi apa yang dapat digunakan untuk membuka berkas tersebut. Ekstensi berkas biasanya ditempatkan di akhir nama file, dipisahkan oleh tanda titik dan juga biasanya terdiri dari dua hingga empat huruf.

8. Bagaimana caranya ini soal simak ui juga 2016​


Jawab:

36

Penjelasan dengan langkah-langkah:

[tex]x^2+ax+b=0[/tex]

Akar"nya

[tex]x_1+x_2=-a\\x_1.x_2=b[/tex]

[tex]x^2+cx+a=0[/tex]

Akar"nya

[tex]x_3+x_4=-c\\x_3.x_4=a[/tex]

Diketahui akar [tex]x_1,x_2[/tex] adalah [tex]\frac{1}{3}[/tex] kali akar [tex]x_3,x_4[/tex]

[tex]x_1=\frac{1}{3} (x_3)[/tex] dan [tex]x_2=\frac{1}{3} (x_4)[/tex]

Sehingga

[tex]x_1+x_2=\frac{1}{3} (x_3)+\frac{1}{3} (x_4)\\x_1+x_2=\frac{1}{3} (x_3+x_4)\\-a=\frac{1}{3} (-c)\\3a=c[/tex]

dan

[tex]x_1.x_2=\frac{1}{3} (x_3).\frac{1}{3} (x_4)\\x_1.x_2=\frac{1}{9}(x_3.x_4)\\b=\frac{1}{9}a\\9b=a[/tex]

Maka;

[tex]\frac{a+c}{b} = \frac{9b+3a}{b} = \frac{9b+3(9b)}{b}= \frac{9b+27b}{b}=\frac{36b}{b}=36[/tex]

21. Bagaimana caranya ini soal simak ui juga 2016​


Jawab:

jika dan hanya jika [tex]a^3=b^2[/tex] atau [tex]a^3=b^4[/tex] (D)

Penjelasan dengan langkah-langkah:

Misal [tex]^alogx=A[/tex] dan [tex]^blogx=B[/tex]

Sehingga,

[tex]9(^alogx)^2+8(^blogx)^2=18(^alogx)(^blogx)\\9A^2+8A^2=18AB\\9A^2-18AB+8B^2=0\\(3A-4B)(3A-2B)=0[/tex]

[tex]3A=4B[/tex] atau [tex]3A=2B[/tex]

*Untuk [tex]3A=4B[/tex]

[tex]3(^alogx)=4(^blogx)\\3^alogx=4^blogx\\^alogx^3= ^blogx^4\\a^3=b^4[/tex]

*Untuk [tex]3A=2B[/tex]

[tex]3(^alogx)=2(^blogx)\\3^alogx=2^blogx\\^alogx^3= ^blogx^2\\a^3=b^2[/tex]

Maka berlaku jika dan hanya jika [tex]a^3=b^2[/tex] atau [tex]a^3=b^4[/tex] (D)

26. tolong di bantu ya soal simak UI



gaya berat
w = m g
w = 3 • 10
w = 30 N

Komponen gaya P
Px = P cos 53° = 0,6 P, ke kanan
Py = P sin 53° = 0,8 P, ke atas

Gaya normal
N = Px
N = 0,6 P, ke kiri

Gaya gesekan
f = u N
f = ¼ • 0,6 P
f = 0,15 P

Nilai P terkecil, saat balok tepat akan bergerak ke bawah. f ke atas
ΣFy = 0
Py + f - w = 0
0,8 P + 0,15 P - 30 = 0
0,95 P = 30
P = 30 / 0,95 ≈ 31,58 N

Nilai P terbesar, saat balok tepat akan bergerak ke atas. f ke bawah
ΣFy = 0
Py - f - w = 0
0,8 P - 0,15 P - 30 = 0
0,65 P = 30
P = 30 / 0,65 ≈ 46,15 N

Jadi balok masih diam, jika nilai P berada dalam selang
31,58 N ≤ P ≤ 46,15 N ← jwb [C]
