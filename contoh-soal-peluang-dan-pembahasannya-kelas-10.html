3. Contoh soal peluang dan pembahasannya


Jawaban:

1. sebuah dadu dilambungkan sekali, tentukan peluang munculnya mata dadu faktor dari 6

jawab:

S = { 1, 2, 3, 4, 5, 6 }, n (S) = 6

Misal nya A = kejadian munculnya mata dadu faktor dari 6

A = { 1, 2, 3 , 6 } , n (A) = 4

Ditanya : P (A) .... ?

P (A) = n (A) / n (S)

= 4 / 6

= 2 / 3

jadi, peluang munculnya mata dadu faktor dari 6 adalah 2/3

 1.Terdapat 1 set kartu bridge sebanyak 52 Buah. Jika seorang mengambil 2 kartu secara acak Dua kali berturut-turut.
Tentukan peluang munculnya satu kartu As dan satu kartu Jack Jika kartu yang sudah di ambil tidak boleh dikembalikan lagi?

Jawab :

Jika A = Kartu pengambilan pertama
Maka P(A) = n(A)/n(S) = 4/52
Jika B|A = Kartu pengambilan kedua setelah kartu pengambilan pertama tidak dikembailikan lagi
Maka P(B | A) = 4/51

Jadi P(A n B) = P(A) x P(B|A)

= 4/52 x 4/51

= 16/2652

= 4/663

29. Contoh soal dan pembahasan limit kelas 10


Jika f(x) = x2 − 6x + 8, tentukan interval f(x) naik dan interval f(x) turun!

Jawab :
f '(x) = 2x − 6

f(x) naik ⇒ f '(x) > 0
⇔ 2x − 6 > 0
⇔ 2x > 6
⇔ x > 3

f(x) turun ⇒ f '(x) < 0
⇔ 2x − 6 < 0
⇔ 2x < 6
⇔ x < 3

Jadi f(x) naik pada interval x > 3 dan turun pada interval x < 3.
