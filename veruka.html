14. konojo wa amerika jin desuka, igirisu jin desukaartikan jepang ke indonesia


Kanojo wa Amerika jin desuka? Igirisu jin desuka? artinya dalam bahasa Indonesia yaitu apakah dia (pr) orang Amerika? Apa orang Inggris?

Pembahasan

彼女はアメリカ人ですか? イギリス人ですか？

Kanojo wa Amerika jin desuka? Igirisu jin desuka?

Apakah dia (pr) orang Amerika? Apa orang Inggris?

_____________

–› 彼女 (kanojo)

Kata ini mengartikan dia (perempuan), bentuk subjek atau kata ganti orang ketiga yang tunggal. Sebenarnya arti kanojo ini juga selain itu bisa diartikan pacar/kekasih, namun karena dalam kalimat diatas sehingga penggunaan kata kanojo diartikan dia.

–› は (wa)

Karena sebelum partikel は (wa) ada subjek kata ganti orang ketiga tunggal yaitu 彼女 (kanojo). Maka dari itu, penggunaan partikel ini berguna agar sebagai penanda subjek.

–› アメリカ (Amerika)

Kata tersebut merupakan nama salah satu negara, karena itu maka penulisan nama negara Amerika ditulis dengan menggunakan huruf katakana.

Huruf-huruf katakana untuk menulis nama negara Amerika :

ア (a).メ (me).リ (ri).カ (ka).

–› 人 (jin)

Ditulis dengan menggunakan kanji, namun kata 人 (jin) ini dibaca secara on'yomi (cara baca China).

Kun'yomi (cara baca Jepang) : ひと (hito).On'yomi (cara baca China) : ニン (nin), dan ジン (jin).

Bentuk kanji hito, nin, dan jin adalah : 人

Artinya yaitu : Orang, manusia.

Penggunaan kata 人 (jin) umumnya untuk memberikan petunjuk atau penjelasan mengenai ciri khas, seperti tempat tinggal/asal/kewarganegaraan orang tersebut.

–› イギリス (Igirisu)

Kata ini juga ditulis dengan menggunakan huruf katakana, dan merupakan nama salah satu negara yaitu Inggris.

Penulisan huruf katakana nama negara Inggris adalah :

イ (i).ギ (gi) : bentuk katakana tenten.リ (ri).ス (su).

–› ですか (desuka)

Partikel pembentuk kalimat tanya.

Arti dari 'Kanojo wa amerika-jin desuka, igirisu-jin desuka' adalah "Apakah dia (perempuan) orang Amerika atau orang Inggris?".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pola Kalimat

== [Subjek] wa [A] desuka, [B] desuka

== Apakah [Subjek] (ini/itu) [A] atau [B]?

Fungsi dari pola kalimat ini adalah untuk menanyakan satu dari dua pilihan. Kalau diartikan secara simpel, seperi "A atau B". Masih ada cara lain untuk membuat kalimat yang serupa, tapi saya hanya akan fokus ke pola kalimat ini saja karena memang ini yang dipakai di soal.

- - -

Untuk pembahasan lebih lanjut, akan saya bahas masing-masing kata yang digunakan untuk membentuk kalimat tersebut.

-

1. 彼女 / かのじょ / Kanojo

Berarti "Dia" atau "Ia" untuk perempuan. Selain itu, kata ini juga dapat diartikan "Pacar (perempuan)". Kata ini umum digunakan sebagai kata ganti orang ketiga tunggal untuk perempuan.

-

2. は / Wa

Tidak ada arti, namun kadang bisa diartikan "Adalah". Kata ini merupakan partikel atau kata bantu kalimat, yang fungsinya untuk menandakan subjek atau topik dari suatu kalimat tersebut. Partikel ini dibaca Wa, tapi penulisannya dalam Hiragana menggunakan Ha.

-

3. アメリカ人 / アメリカじん / Amerika-jin

Berarti "Orang Amerika". Terdiri dari kata Amerika (Amerika) dan Jin (Orang). Kata Jin setelah kata benda berfungsi untuk mendeskripsikan keadaan atau sifat seseorang. Pada kata ini, kata benda nya adalah Amerika (Amerika), jadi bila digabung dengan Jin (Orang) akan menjadi "Orang Amerika".

-

4. です / Desu

Tidak ada arti. Merupakan akhiran kalimat yang berfungsi untuk memperhalus dan mempertegas suatu kalimat tersebut. Kata ini diletakkan setelah predikat, tapi tidak bisa setelah kata kerja. Untuk kata kerja, dengan mengubahnya ke bentuk Masu. Adapun kata serupa yang sering dianggap sebagai 'bentuk kasual' dari Desu, yaitu 'Da'. Kata Da ini hanya mempertegas kalimat saja, tidak memperhalus.

-

5. か / Ka

Tidak ada arti, tapi perannya sama seperti tanda tanya. Kata ini merupakan partikel yang berfungsi untuk mengubah suatu kalimat biasa menjadi kalimat tanya. Biasa diletakkan setelah 'Desu / Masu'.

-

6. イギリス人 / イギリスじん / Igirisu-jin

Berarti "Orang Inggris". Terdiri dari kata Igirisu (Inggris) dan Jin (Orang). Untuk penjelasan mengenai kata Jin, bisa dilihat di nomor 3.

-

Kata-kata di atas bila digabung akan menjadi 彼女はアメリカ人ですか、イギリス人ですか (Kanojo wa amerika-jin desuka, igirisu-jin desuka), yang berarti "Apakah dia (perempuan) orang Amerika atau orang Inggris?".

15. apakah anda tahu tentang ceruta pendek


Cerpen adalah cerita pendek atau cerita yg diringkas dgn bahasa sendiri dgn tanpa mengubah isi daricerita. Tema,Alur cerita,Tokoh cerita,Watak tokoh,Amanat..Maaf ya kalau salahCerita pendek atau sering disingkat sebagai cerpen adalah suatu bentuk prosanaratif fiktif. Cerita pendek cenderung padat dan langsung pada tujuannya dibandingkan karya-karya fiksi lain yang lebih panjang, seperti novella (dalam pengertian modern) dan novel. Karena singkatnya, cerita-cerita pendek yang sukses mengandalkan teknik-teknik sastra seperti tokoh, plot, tema, bahasadan insight secara lebih luas dibandingkan dengan fiksi yang lebih panjang. Ceritanya bisa dalam berbagai jenis.

Cerita pendek berasal dari anekdot, sebuah situasi yang digambarkan singkat yang dengan cepat tiba pada tujuannya, dengan paralel pada tradisi penceritaan lisan. Dengan munculnya novel yang realistis, cerita pendek berkembang sebagai sebuah miniatur, dengan contoh-contoh dalam cerita-cerita karyaE.T.A. Hoffmann dan Anton Chekhov.
