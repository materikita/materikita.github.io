5. Hitunglah jumlah deret aritmatika 124+120+116+....+0

Bab Barisan dan Deret Bilangan

b = U2 - U1
b = 120 - 124
b = -4

Un = 0
a + (n - 1) . b = 0
124 + (n - 1) . (-4) = 0
(n - 1) . (-4) = -124
n - 1 = -124/-4
n = 31 + 1
n = 32

Sn = n/2 . (a + Un)
S32 = 32/2 . (124 + 0)
S32 = 16 . 124
