Present Simple Tense digunakan untuk menyatakan:

- Kejadian yang terjadi pada saat ini

- Kebenaran umum

- Hobi

Present Simple Tense menggunakan Verb 1 atau Verb Infinitive.

Rumus Present Simple Tense:

Untuk Subject I, You, We, They

(+) S + V1 + O

(-) S + 'don't' + V1 + O

(?) 'Do' + S + V1 + O + ?

Untuk Subject He, She, It

(+) S + (V1 + s/es) + O

(-) S + 'doesn't' + V1 + O

(?) 'Does' + S + V1 + O + ?

Keterangan:

(+) = Kalimat positif (positive)

(-) = Kalimat negatif (negative)

(?) = Kalimat tanya (interrogative/question)

S = Subjek/Kata ganti orang (subject)

V1 = Verb 1

O = Objek/Kata benda (Objek)
