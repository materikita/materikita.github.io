14. Pak Yuan bekerja di suatu perusahaan dengan gaji pada tahun pertama bekerja adalah Rp4.000.000,00 per bulan, dimana setiap tahun gaji Pak Yuan akan meningkat sebesar Rp400.000,00. Setiap bulan Pak Yuan berencana menyisihkan 20% dari gaji yang diperolehnya untuk ditabung. Jika tabungan tersebut tidak pernah diambil, maka jumlah tabungan Pak Yuan setelah 5 tahun adalah …​


Jawab:

Rp.57.600.000

Penjelasan dengan langkah-langkah:

Gaji tahun pertama = Rp.4.000.000

Simpanan 1 = 20% x Rp.4.000.000 x 12 bulan = Rp.9.600.000

Gaji tahun kedua = Rp.4.000.000 + Rp.400.000 = Rp.4.400.000

Simpanan 2 = 20% x Rp.4.400.000 x 12 bulan = Rp.10.560.000

Gaji tahun ketiga = Rp.4.400.000 + Rp.400.000 = Rp.4.800.000

Simpanan 3 = 20% x Rp.4.800.000 x 12 bulan = Rp.11.520.000

Gaji tahun ketiga = Rp.4.800.000 + Rp.400.000 = Rp.5.200.000

Simpanan 4 = 20% x Rp.5.200.000 x 12 bulan = Rp.12.480.000

Gaji tahun ketiga = Rp.5.200.000 + Rp.400.000 = Rp.5.600.000

Simpanan 5 = 20% x Rp.5.600.000 x 12 bulan = Rp.13.440.000

Total Tabungan Pak Yuan Selama 5 Tahun

= Rp.9.600.000 + Rp.10.560.000 + Rp.11.520.000 + Rp.12.480.000 + Rp.13.440.000

= Rp.57.600.000

31. -20 - 580 =-20 - (-580) =​


Jawaban:

= -20 - 580

= -20 - (-580)

= -600

Pembahasan :

Dalam penjumlahan bilangan bulat, Perlu diketahui

a) (+) + (+) = (+)

Contoh: 3 + 9 = 12

b) (-) + (-) = (-)

Contoh: ( -2) + (-3) = (-5)

c) (+) + (-) = bisa (+) bisa juga (-)

Jika (+) lebih besar maka hasilnya (+)

Jika (-) lebih besar maka hasilnya (-)

Contoh:  (-5) + 2 = (-3) → hasilnya negatif    

              (-5) + 9 = 4 → hasilnya positif

Dalam perkalian bilangan bulat, perlu diketahui juga :

(-) × (-) = (+)

(+) × (-) = (-) (-) × (+) = (-)

Penyelesaian soal :

a) -20 - 580 =

⇒ - 600

b) -20 - (-580) =​

⇒ - 20 + 580

⇒ 560
