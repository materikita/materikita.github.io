4. Mencari rumus beserta contoh soalnya pada cermin datar, cekung, dan cembung​


Jawaban:

Tentu! Berikut ini adalah rumus dan contoh soal untuk cermin datar, cekung, dan cembung:

1. Cermin Datar:

- Rumus: s = -s'

Keterangan:

- s = jarak objek ke cermin (positif jika objek berada di depan cermin, negatif jika objek berada di belakang cermin)

- s' = jarak bayangan ke cermin (positif jika bayangan ada di depan cermin, negatif jika bayangan ada di belakang cermin)

Contoh Soal: Sebuah objek ditempatkan 20 cm di depan cermin datar. Tentukan posisi dan sifat bayangan yang terbentuk jika menggunakan cermin datar.

2. Cermin Cekung:

- Rumus: 1/f = 1/s + 1/s'

Keterangan:

- f = jarak fokus cermin (positif untuk cermin cekung)

- s = jarak objek ke cermin (positif jika objek berada di depan cermin, negatif jika objek berada di belakang cermin)

- s' = jarak bayangan ke cermin (positif jika bayangan ada di depan cermin, negatif jika bayangan ada di belakang cermin)

Contoh Soal: Sebuah objek ditempatkan 30 cm di depan cermin cekung dengan jarak fokus 15 cm. Hitunglah posisi dan sifat bayangan yang terbentuk menggunakan cermin cekung.

3. Cermin Cembung:

- Rumus: 1/f = 1/s - 1/s'

Keterangan:

- f = jarak fokus cermin (negatif untuk cermin cembung)

- s = jarak objek ke cermin (positif jika objek berada di depan cermin, negatif jika objek berada di belakang cermin)

- s' = jarak bayangan ke cermin (positif jika bayangan ada di depan cermin, negatif jika bayangan ada di belakang cermin)

Contoh Soal: Sebuah objek ditempatkan 40 cm di depan cermin cembung dengan jarak fokus -20 cm. Tentukan posisi dan sifat bayangan yang terbentuk menggunakan cermin cembung.

Perhatikan bahwa dalam contoh soal di atas, nilai jarak objek (s) dan jarak bayangan (s') diberikan dengan tanda positif atau negatif sesuai dengan konvensi yang digunakan dalam fisika cermin.

Pembahasan

Diketahui :

Panjang fokus (f) = 15 cm

Panjang fokus lensa cembung bertanda positif karena titik fokus lensa cembung dilalui berkas cahaya atau bersifat nyata.

Tinggi benda (h) = 5 cm

Jarak benda (s) = 5 cm

Ditanya : jarak bayangan, perbesaran bayangan, tinggi bayangan dan sifat bayangan

Jawab :

Gambar pembentukan bayangan benda oleh lensa cembung :

Contoh soal lensa cembung - 1

Jarak bayangan (s’) :

1/s’ = 1/f – 1/s = 1/15 – 1/5 = 1/15 – 3/15 = -2/15

s’ = -15/2 = -7,5 cm

Jarak bayangan bertanda negatif artinya bayangan bersifat maya atau bayangan tidak dilewati berkas cahaya.

Jarak bayangan 7,5 cm lebih besar daripada jarak benda 5 cm.

Perbesaran bayangan (M) :

M = -s’/s = -(-7,5)/5 = 7,5/5 = 1,5 kali

Perbesaran bayangan bertanda positif artinya bayangan tegak atau tidak terbalik.

Ukuran bayangan 1,5 kali lebih besar daripada ukuran benda.

Tinggi bayangan (h’) :

M = h’/h

h’ = M h = (1,5)5 = 10/3 = 7,5 cm

Tinggi bayangan bertanda positif artinya bayangan tegak atau tidak terbalik.

Tinggi bayangan 7,5 cm lebih besar daripada tinggi benda 5 cm.

Sifat bayangan

Berdasarkan perhitungan di atas disimpulkan sifat-sifat bayangan benda yang dibentuk oleh lensa cekung adalah :

– maya atau berkas cahaya tidak melewati bayangan

– Tegak atau tidak terbalik

– Diperbesar atau ukuran bayangan lebih kecil daripada ukuran benda

– Jarak bayangan lebih besar daripada jarak benda

Panjang fokus (f) = -30 cm

Jarak bayangan (s’) :

s’ = -30/3 = -10 cm

M = h’/h
h’ = M h = (2/3)5 = 10/3 = 3,3 cm
Tinggi bayangan bertanda positif artinya bayangan tegak atau tidak terbalik.
Tinggi bayangan 3,3 cm lebih kecil daripada tinggi benda 5 cm.

15. Diberikan fungsi = −2 3 + 3 2 pada interval [−1, 1] 1. Tentukan titik kritis. 2. Tentukan nilai maksimum dan minimum. 3. Tentukan di mana monoton naik dan di mana monoton turun.4. Tentukan di mana fungsi cekung ke atas atau cekung ke bawah. 5. Tentukan titik balik.​


Jawab:

Untuk menyelesaikan masalah ini, kita akan melakukan analisis terhadap fungsi f(x) = -2x^3 + 3x^2 pada interval [-1, 1].

1. Titik Kritis:

  Titik kritis terjadi ketika turunan pertama fungsi f(x) sama dengan nol. Jadi, kita akan mencari turunan pertama:

  f'(x) = -6x^2 + 6x

  Untuk mencari titik kritis, kita set f'(x) sama dengan nol dan selesaikan persamaan:

  -6x^2 + 6x = 0

  Faktorkan -6x:

  -6x(x - 1) = 0

  Dari sini, kita memiliki dua solusi:

  a. x = 0

  b. x = 1

  Jadi, titik kritis terletak di x = 0 dan x = 1.

2. Nilai Maksimum dan Minimum:

  Untuk menentukan nilai maksimum dan minimum, kita perlu menguji nilai fungsi di titik kritis dan di ujung interval [-1, 1].

  a. Di x = -1:

     f(-1) = -2(-1)^3 + 3(-1)^2 = -2 - 3 = -5

  b. Di x = 0:

     f(0) = -2(0)^3 + 3(0)^2 = 0

  c. Di x = 1:

     f(1) = -2(1)^3 + 3(1)^2 = -2 + 3 = 1

  Jadi, nilai minimum adalah -5 dan nilai maksimum adalah 1.

3. Monoton Naik dan Monoton Turun:

  Untuk menentukan di mana fungsi monoton naik dan turun, kita perlu memeriksa tanda turunan pertama, f'(x).

  a. Di interval [-1, 0]:

     -1 ≤ x ≤ 0, f'(x) = -6x^2 + 6x selalu positif. Jadi, f(x) monoton naik di interval ini.

  b. Di interval [0, 1]:

     0 ≤ x ≤ 1, f'(x) = -6x^2 + 6x selalu negatif. Jadi, f(x) monoton turun di interval ini.

4. Fungsi Cekung ke Atas atau Cekung ke Bawah:

  Untuk menentukan di mana fungsi cekung ke atas atau ke bawah, kita perlu memeriksa tanda turunan kedua, f''(x).

  f''(x) = -12x + 6

  a. Di x = 0:

     f''(0) = -12(0) + 6 = 6, positif. Jadi, fungsi cekung ke atas di x = 0.

5. Titik Balik:

  Titik balik terjadi ketika turunan kedua fungsi f(x) berubah tanda. Kami telah menentukan bahwa f''(0) positif (cekung ke atas) pada x = 0. Oleh karena itu, x = 0 adalah titik balik.

Jadi, untuk fungsi f(x) = -2x^3 + 3x^2 pada interval [-1, 1]:

1. Titik kritis terletak di x = 0 dan x = 1.

2. Nilai minimum adalah -5 dan nilai maksimum adalah 1.

3. Fungsi monoton naik di interval [-1, 0] dan monoton turun di interval [0, 1].

4. Fungsi cekung ke atas di sekitar x = 0.

5. Titik balik terletak di x = 0.

18. 1. Diketahui fungsi (x-c)²+c, dimana c = 26Tentukana. Titik-titik Stasioner fungsib. selang monoton naik dan monton turunc. Titik belok/balikd. selang kecekungan (cekung ke atas)e. selang kecekungan (cekung ke bawah)f. Sketsa grafik fungsi​


Misalkan [tex] f(x) = (x - c)^2 + c[/tex]

(a) Titik stasioner, titik kritis, titik ekstrim

Untuk mencari titik stasioner/titik kritis/titik ekstrim, kita harus mencari nilai [tex] x [/tex] dengan turunan fungsi pertama sama dengan nol.

[tex] \begin{align} f^{\prime}(x) &= 0 \\ 2(x - c) &= 0 \\ x &= c \end{align} [/tex]

Jadi,

[tex] f(c) = c [/tex]

(b) Selang monoton turun dan naik

Mirip dengan cara sebelumnya.

Untuk selang menurun

[tex] f^{\prime}(x) < 0 \Leftrightarrow x < c [/tex]

Untuk selang menaik

[tex] f^{\prime}(x) > 0 \Leftrightarrow x > c [/tex]

(c) Titik balik

Untuk titik balik, kita harus mencari nilai [tex] x [/tex] ketika turunan fungsi kedua sama dengan nol. Jika tidak ada, maka tidak ada titik belok. Begitu juga sebaliknya.

[tex] \begin{align} f^{\prime \prime}(x) &= 0 \\ \frac{\mathrm{d}}{\mathrm{d}x} \left[ 2x - 2c \right] &= 0 \\ 2 &\ne 0 \end{align} [/tex]

Disini tidak ada nilai [tex] x [/tex], maka tidak terdapat titik belok.

(d) Selang kecekungan (cekung ke atas)

Ini mirip dengan cara sebelumnya, namun turunan fungsi kedua harus lebih kecil daripada nol.

[tex] f^{\prime \prime}(x) < 0 \Leftrightarrow 2 \nless 0 [/tex]

Tidak ada fungsi yang cekung ke atas di [tex] x \in \mathbb{R} [/tex].

(e) Selang kecekungan (cekung di bawah)

Mirip dengan cara sebelumnya, namun turunan fungsi kedua lebih besar daripada nol.

[tex] f^{\prime \prime}(x) > 0 \Leftrightarrow 2 > 0 [/tex]

Yang berarti, fungsi ini cekung ke bawah untuk [tex] x \in \mathbb{R} [/tex]
