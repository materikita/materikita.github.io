- Jumlah deposito awal (P) = Rp 800.000,-

- Tingkat bunga deposito (r) = 8% per tahun

Rumus bunga majemuk:

A = P * (1 + r)^n

Dimana:

- A adalah nilai akhir investasi

- P adalah jumlah deposito awal

- r adalah tingkat bunga deposito (dalam desimal)

- n adalah jangka waktu (dalam tahun)

Mari kita hitung nilai investasi pada akhir tahun pertama, kedua, dan seterusnya hingga tahun ke-50.

1. Untuk tahun pertama (n = 1):

A1 = 800.000 * (1 + 0.08)^1

= 800.000 * 1.08

= 864.000

2. Untuk tahun kedua (n = 2):

A2 = 800.000 * (1 + 0.08)^2

= 800.000 * 1.1664

= 933.120

3. Untuk tahun ketiga (n = 3):

A3 = 800.000 * (1 + 0.08)^3

= 800.000 * 1.2597

= 1.007.760

Kita dapat melanjutkan perhitungan ini hingga tahun ke-50 dengan menggunakan rumus yang sama.
