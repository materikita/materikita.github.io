Penjelasan:

◆ Structure kalimat diatas merupakan Conditional sentence tipe 2 .

◆Untuk meng-identifikasi jenis structure nya , harap diperhatikan kata kerja pada klausa kedua.

◆ Ada 3 jenis Conditional sentence ( kalimat andai-andai )

¶ Conditional type 1 { Truth in the present/future }

~Menyatakan sebuah kondisi nyata yang akan dilakukan saat ini atau nanti.

RUMUS TYPE 1 :

IF+S+VERB (S/ES)+ O/C , SUBJECT+WILL+VERB 1 + O/C

~Example :

If she HAS enough money, she WILL BUY a new book later.

¶ Conditional type 2 { Untruth in the present / future }

~ Menyatakan kondisi yang berbanding terbalik dengan kenyataannya ( untuk saat ini atau mendatang )

RUMUS TYPE 2 :

IF+S+VERB 2 + O/C , S + WOULD + VERB 1 + O/C

~ Example :

If I WERE a bird , I WOULD FLY around the world .

{ Kebohongan saat ini Dan mendatang karena tidak mungkin terwujud }

¶ Conditional type 3 { Untruth in the past }

~ Menyatakan penyesalan atau kondisi yang tidak terwujud di waktu lampau.

RUMUS TYPE 3 :

IF+S+HAD+VERB3+O/C , S+WOULD+HAVE + VERB 3 + O/C

Example :

If I HAD HAD much money, I WOULD HAVE BOUGHT a new car last year .

{ Kenyataanya di waktu lampau " I didn't buy a new car because I didn't have much money " }

Note: Ini adalah contoh if clause/conditional type 1.

Conditional type one menggambarkan konsekwensi yang mungkin terjadi nanti/di masa datang

Kalau kita pilah kalimatnya:

Animals will run away (konsekwensi yang mungkin terjadi nantinya. Perhatikan ini kemungkinan yg mungkin terjadi, bukan khayalan atau angan-angan)

if (jika)

A volcano explodes (suatu kondisi yang terjadi)

Rumus conditional type 1:

simple future (dalam hal ini animals will run away) + if + simple present (dalam hal ini a volcano explodes)

Kalimat di atas juga bisa kita susun secara berbeda dengan makna yang sama, yaitu dengan meletakkan If  di awal kalimat:

If a volcano explodes, Animals will run away

dalam hal ini rumusnya adalah:

if + simple present, simple future tense
