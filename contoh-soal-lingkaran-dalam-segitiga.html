1. Contoh soal dan jawaban lingkaran dalam segitiga


Diketahui panjang sisi-sisi sebuah segitiga adalah 23 cm, 27, dan 32 cm. Hitunglah jari-jari lingkaran dalam segitiga tersebut!

Misalkan a = 23, b = 27, dan c = 32

s = ½ keliling segitiga

s = ½ (a + b + c)

s = ½ (23 + 27 + 32)

s = 41 cm


L Δ = √(s(s-a)(s-b)(s-c))

L Δ = √(41(41-23)(41-27)(41-32))

L Δ = √(41(18)(14)(9))

L Δ = √92988

L Δ = 304,94 cm2


r = L Δ/s

r = 304,94 cm2/41 cm

r = 7,4 cm
