2. If had found her adress, I ... (Send) her invitation


Answer :

If I had found her address, I would have sent her an invitation.

Kalimat tersebut menggunakan konstruksi conditional sentence type 3.

----------

Explanation :

Conditional sentence adalah kalimat pengandaian. Ada tiga tipe kalimat pengandaian :

Type 1 → untuk kejadian yang masih mungkin terjadi di masa yang akan datang (future possible).

Construction : IF + S + infinitive/infinitive+s, S + will + infinitive


Type 2 → pengandaian untuk peristiwa yang terjadi di masa sekarang (present unreal).

Construction : IF + S + preterite (v₂), S + would + infinitive (v₁)


Type 3 → pengandaian untuk peristiwa yang terjadi di masa lampau (past unreal).

Construction : IF + S + had + past participle (v₃), S + would + have + past participle (v₃)

-----------

Answer Details

Subject : English

Category : Structure and written expressions

Sub-category : Conditional clause

Level : JHS VIII

Code : 8.5

Keywords : if, type 2, conditional

---------

If had found her adress, I *would have Sent* her invitation

Rumus Conditional Sentences Type 3
If + Subject + had + verb (past perfect tense) + Subject + would + have + verb (past future perfect tense)

*send v1 - sent v2 - sent v3

Penjelasan:

Jawaban dicetak tebal dan merupakan materi pembelajaran bentuk kalimat Simple Past Tense karena menjelaskan aktivitas kegiatan di masa lampau. Pilihan lainnya salah.

Simple Past Tense:

(+) Subject + Verb-2 + Object

Semoga membantu ya.

Tips

Pada simple present tense, kata kerja dengan subjek he/she/it selalu berakhiran dengan imbuhan tambahan seperti -s, -es, atau -ies. Contoh:

It walks ...She writes ...He watches ...She finds ....He looks ...It flies ...

Pada simple past tense, tidak ada tambahan imbuhan apapun. Semuanya sama untuk subjek apa pun. Contoh:

It walked ...You walked ...She walked ...They walked ...He walked ...It walked ...I walked ...

Jawaban

She (find/found) an enormous cucumber in her field.

(Dia menemukan sebuah mentimun besar di sawahnya.)

Subjek she, kata kerja present tense harus berimbuhan -sKarena hanya ada find tanpa imbuhan -s (seharusnya finds), maka jelas bukan itu jawabannyaJadi jawabannya adalah yang satunya, yaitu found (simple past)
