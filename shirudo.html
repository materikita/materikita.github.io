5. Hirudo merupakan jenis cacing parasit yang mengisap darah dari tubuh inangnya . Bagaimana cara hidup Hirudo?


Hirudo Hewan berhabitat air tawar, hidup di rawa-rawa, kolam, ataupun sungai. Saat merobek atau membuat lubang, lintah mengeluarkan zat anestetik (penghilang sakit), sehingga korbannya tidak akan menyadari adanya gigitan. Setelah ada lubang, lintah akan mengeluarkan zat anti pembekuan darah yaitu hirudin. Dengan zat tersebut lintah dapat mengisap darah sebanyak mungkin.

Pendahuluan :

Kata sifat dalam Bahasa Jepang disebut dengan 形容詞 (Keiyoushi). Kata sifat Bahasa Jepang dibagi menjadi kata sifat-i dan kata sifat-na. Perbedaannya yaitu kata sifat-i menggunakan tambahan huruf i untuk menerangkan kata benda, sedangkan kata sifat-na menggunakan tambahan huruf na. Namun terdapat beberapa kata sifat yang menggunakan tambahan huruf i, tapi termasuk ke dalam kata sifat-na. Salah satu contohnya yaitu Kirai (Benci).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

白い (Shiroi) memiliki arti "Putih", dan kata ini termasuk ke dalam kelompok kata sifat-i. Kata sifat-i merupakan kelompok kata sifat yang terdapat tambahan huruf i di akhir kata nya. Contohnya seperti kata Shiroi, yang awalnya hanya Shiro kemudian ditambahkan huruf i. Menambahkan huruf i di akhir kata sifat-i ini fungsinya adalah untuk menerangkan kata benda nya. Contoh kalimatnya sebagai berikut.

白い鞄 (Shiroi kaban) = Tas putih白い服 (Shiroi fuku) = Baju putih白い髪 (Shiroi kami) = Rambut putih

Beberapa kata sifat-i jika tidak terdapat tambahan huruf i nya, akan menjadi kata benda. Contohnya seperti kata Shiroi ini. Shiroi merupakan kata sifat, dan Shiro merupakan kata benda. Contoh kalimat yang menggunakan kata Shiro sebagai berikut.

私の鞄は白 (Watashi no kaban wa shiro) = Tas saya putih私の服は白 (Watashi no fuku wa shiro) = Baju saya putih私の髪は白 (Watashi no kami wa shiro) = Rambut saya putih

31. sebutkan filum taenia solium, acatina fulica, dan obelia hirudo medicinalis!​


Taenia solium atau cacing pipih termasuk ke dalam filum Platyhelminthes.Achatina fulica atau bekicot termasuk ke dalam filum Mollusca.Obelia hirudo medicinali atau lintah termasuk ke dalam filum Cnidaria.

Pembahasan

Platyhelminthes merupakan salah satu filum dari kingdom animalia (binatang). Filum ini berisi semua cacing pipih kecuali Nemertea. Tubuhnya datar dorsoventral dan tidak tersegmentasi. Cacing pipih umumnya hidup di sungai, danau, lautan, atau sebagai parasit pada organisme lain. Cacing dalam kelompok ini sangat sensitif terhadap cahaya.

Moluska adalah hewan triploblastik selomata yang tubuhnya lunak. Tubuh tidak tersegmentasi, simetris. Tubuh terdiri dari "kaki" berotot dan kepala yang berkembang bervariasi dari kelas ke kelas. Kaki digunakan untuk beradaptasi, bertahan di pangkalan, menggali dan menembus pangkalan, berenang, atau bergerak.

Cnidaria adalah filum sederhana yang hanya ditemukan di perairan, terutama lingkungan laut. Ciri khas cnidaria adalah cnidaria, yang merupakan sel khusus yang digunakan terutama untuk menangkap  dan mempertahankan mangsa. Tubuh mereka terbuat dari mesoglea, zat agar-agar tidak hidup yang biasanya diapit di antara dua lapisan epitel yang  setebal satu sel. Mereka memiliki dua bentuk tubuh dasar: medusa yang berenang bebas dan polip sesil, keduanya simetris radial dengan mulut dikelilingi oleh tentakel knidosit.

39. Perhatikan beberapa contoh penulisan nama ilmiah berikut! 1) Ascaris Lumbricoides 2) Pinctada maxima 3) hirudo medicinalis


Penulisan nama ilmiah diatur dalam sistem binominal nomenklatur. Sistem binomial nomenklatur merupakan sistem pemberian nama hewan atau tumbuhan secara sah dan benar berdasar kode internasional.  Pemberian nama ini diatur dengan Kode Internasional Tata Nama Hewan  dan Tumbuhan dengan menggunakan sistem tata nama dua kata (binomial  nomenklatur).

Pada penulisan nama ilmiah:

1. Ascaris lumbricoides adalah nama ilmiah untuk hewan cacing gelang yang merupakan parasit pada tubuh manusia karena dapat menyebabkan penyakit Askariasis. Pada penamaan ilmiah tersebut dapat diketahui bahwa Askaris merupakan nama genus sedangkan lumbricoides merupakan nama spesies.

2. Pinctada maxima adalah nama ilmiah untuk hewan kerang mutiara yang bermanfaat bagi manusia karena menghasilkan mutiara yang memilki nilai ekonomi tinggi. Budidaya kerang mutiara di Indonesia banyak di daerah Proponsi Nusa Tenggara Barat. Berdasarkan penamaan ilmiah tersebut pada Pinctada merupakan nama genus dan maxima merupakan nama spesies.

3. Hirudo medicinalis merupakan nama ilmiah untuk lintah, yang dimasukkan ke dalam kelompok cacing. Meskipun lintah hidup di rawa dan menghisap darah tetapi lintah dapat dimanfaatkan manusia untuk pengobatan akupuntur / bekam. Berdasarkan nama ilmiah tersebut penamaan Hirudo termasuk genus dan pada medicinalis termasum nama spesies.


PEMBAHASAN LEBIH LANJUT:

Sistem pemberian nama makhluk hidup yang digunakan Linnaeus disebut Sistem Binomial Nomenklatur dan bahasa yang digunakan adalah bahasa Latin. Dengan demikian, untuk suatu macam makhluk hidup hanya digunakan satu nama bagi seluruh dunia ilmu pengetahuan.

Dengan adanya kesatuan nama ini, orang tidak akan keliru dengan makhluk hidup yang dimaksud meskipun di tiap negara atau daerah memiliki nama sendiri. Sistem binomial nomenklatur ini merupakan sistem pemberian nama hewan atau tumbuhan secara sah dan benar berdasar kode internasional.

Pemberian nama ini diatur dengan Kode Internasional Tata Nama Hewan dan Tumbuhan dengan menggunakan sistem tata nama dua kata (binomial nomenklatur) dengan aturan-aturan sebagai berikut:

1. Menggunakan bahasa latin/Itali.

2. Nama terdiri dari dua kata, kata pertama menunjukkan tingkatan marga (genus) yang diawali dengan huruf besar. Kata kedua menunjukkan tingkatan jenis (spesies) yang diawali dengan huruf kecil.

Contohnya: Gnetum gnemon

3. Jika ditulis dengan huruf tegak, dua kata tersebut harus digarisbawahi, tetapi jika tidak digaris bawahi, dua kata tersebut harus dicetak miring atau dicetak tebal. Contohnya: Gnetum gnemon, Gnetum gnemon atau Gnetum gnemon.

Jika memiliki subspesies, nama tersebut ditambahkan pada kata ketiga. Jadi, pada subspesies terdiri atas tiga kata. Sistem penamaan yang terdiri atas tiga suku kata disebut Trinomial, contohnya Passer domesticus domesticus (burung gereja) dan Felis maniculata domesticus (kucing jinak).

Untuk kelompok yang tingkatan klasifikasinya lebih tinggi lagi, aturan penamaannya adalah sebagai berikut:

a. Pada hewan

- Nama famili berasal dari nama genus ditambah idae.

Contoh: Ranidae berasal dari Rana (katak).

- Nama subfamili berasal dari nama genus, ditambah inae.

Contoh: Fasciolinae berasal dari Fasciola (cacing pita).

b. Pada tumbuhan

- Nama famili diberi akhiran aceae atau ae.

Contoh: Ranunculaceae berasal dari Ranunculus, Leguminoceae berasal dari Leguminose.

- Nama ordo diberi akhiran ales.

Contoh: Filiales (paku-pakuan).

- Nama divisio diberi akhiran phyta.

Contoh: Spermatophyta.
