21. Susi :.......Santi : Santi desubahasa Jepang​


Susi : あなたの名前はなんですか? Atau

( Anata no Namae wa nan desu ka ? ) → Nama anda siapa ?

お名前は何ですか? Ataupun

( O Namae wa nan desu ka ? ) → Nama anda siapa ?

君の名は?

( Kimi no nawa ? ) → Namamu ?

Santi : サンチです

( Sanchi desu ) → Santi

Pembahasan

Secara umum biasanya menanyakan Nama dalam bahasa Jepang , anda biasa menyebutkan :

- お名前は ( O Namae wa )

- あなたの名前はなんですか ( Anata no Namae wa nan desu ka )

Jika anda Ingin bertanya sesuatu biasanya pada kalimat yang anda Sebutkan akan ada Kata ‘か’ ( Ka : Merupakan Partikel Tanya ) atau ですか ( Desu ka )

あなたの名前はなんですか? Atau

( Anata no Namae wa nan desu ka ? ) → Nama anda siapa ?

Terjemah Per-Kata;

あなた ( Anata ) → Memiliki arti "Anda''

の ( No ) → Merupakan Partikel yang menunjukkan Kepemilikan

名前 ( Namae ) → Memiliki arti "Nama"

は ( Wa ) → Partikel yang menunjukkan Subjek

何 ( Nan ) → Memiliki arti "Apa''

です ( Desu ) → Tidak Memiliki arti berfungsi memperhalus kalimat lain

か ( Ka ) → Partikel tanya

お名前は何ですか? Ataupun

( O Namae wa nan desu ka ? ) → Nama anda siapa ?

Terjemah Per-Kata;

お ( O ) → Disini berfungsi sebagai kalimat yang menunjukkan kesopanan kepada orang yang belum dikenal

名前 ( Namae ) → Memiliki arti nama

は ( Wa ) → Partikel yang menunjukkan Subjek

何 ( Nan ) → Memiliki arti "Apa''

です ( Desu ) → Tidak Memiliki arti berfungsi memperhalus kalimat lain

か ( Ka ) → Partikel tanya

君の名は?

( Kimi no nawa ? ) → Namamu ?

Terjemah Per-Kata;

君 ( Kimi ) → Memiliki arti "Kamu"

の ( No ) → Partikel yang menunjukkan Kepemilikan

名 ( Na ) → Namae Memiliki arti "Nama" Na sendiri sudah mewakili arti dari Namae

は ( Wa ) → Partikel yang menunjukkan Subjek

Pada dasarnya menyatakan nama Dalam bahasa Jepang paling sering digunakan adalah ‘Watashi no Namae’ , Merupakan Pernyataan paling mudah untuk dipahami , Watashi no Namae Sendiri Memiliki Arti yaitu “Nama Saya adalah’’

cara untuk memperkenalkan diri dalam bahasa Jepang juga cukup mudah terpenting adalah anda tahu cara dan rumus untuk memperkenalkan diri :

私 + の + 名前 + は + [ Nama ] + です

[ Watashi no Namae wa ( Nama ) desu ]

--> Nama saya adalah [ Nama ]

私 + [ Nama ] + です

[ Watashi ( Nama ) Desu ]

--> Saya adalah [ Nama ]

anda tinggal isi oleh nama anda sendiri pada kolom yang diberikan tanda kurung

サンチです ( Sanchi desu ) → Santi , Kalimat ini menerangkan tentang kepemilikan nama , untuk memperkenalkan diri dengan lebih sopan dalam bahasa Jepang anda bisa mengucapkan;

私の名前はサンチです ( Watashi No Namae wa Sanchi desu ) → Nama saya santi.

私 [ Watashi ] :

Watashi Memiliki arti yaitu “Saya” , Watashi Sangat Sering digunakan , Pada Dasarnya Watashi Digunakan Oleh Pria Maupun Wanita

の [ No ] :

の Merupakan partikel yang menggabungkan antara kata 1 dan kata 2 , Partikel no juga merupakan Partikel yang Menunjukkan Kepemilikan

名前 [ Namae ] :

Namae atau “名前” Memiliki arti Yaitu ‘Nama’ dalam bahasa Indonesia , atau ‘Name’ yang dalam Bahasa Inggris nya . 名 ( Na ) Sendiri sudah mewakili arti kalimat ‘nama’

は [ Wa → わ ] :

は Merupakan Partikel yang Menunjukkan Subjek dalam suatu kalimat , は ditulis ‘ha’ tetapi Di artikan (dibaca) sebagai ‘Wa’, Partikel wa biasanya diikuti dengan kata ‘です’ ( desu )

です [ Desu ] :

Desu Merupakan Kalimat Penghalus , Kalimat tersebut Guna nya Untuk Memperhalus kalimat Lain agar menjadi kalimat Yang lebih sopan , Kata desu biasanya di letakkan di akhir kalimat

23. Santi membeli baju 150,000,00 pada saat membayar di kasir santi mendapatkan diskon 15% berapa rupiah yang harus d bayarkan santi


Jawaban:

127.000,00

Penjelasan dengan langkah-langkah:

150.000 - 15% = 127,000

Semoga Terbantu

Jawaban:

Diketahui:

Santi membeli baju 150,000,00

mendapatkan diskon 15%

Ditanya:

berapa rupiah yang harus d bayarkan santi

Penjelasan dengan langkah-langkah:

mencari diskon

rumus:

diskon = presentase diskon × harga beli

[tex]15\% \times 150.000 = \frac{15}{100} \times 150.000 \\ = \frac{2.250.000}{100} \\ = 22.500[/tex]

uang yg harus dibayar santi

rumus:

harga beli - diskon

= 150.000 - 22.500

= 127.500

jadi uang yg harus dibayar santi sebesar 127.500
