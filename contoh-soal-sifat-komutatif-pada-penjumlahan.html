13. tulislah contoh operasi hitung yang memenuhi sifat komutatif penjumlahan dan sifat asosiatif penjumlahanplis bantu kumohon pliss​


✏Pembahasan

• Sifat komutatif disebut juga sifat pertukaran. Sifat komutatif yaitu :

• a + b = b + c

• Sifat asosiatif disebut juga sifat pengelompokan. Pada sifat asosiatif, bilangan-bilangan dapat dikelompokkan sesuai yang akan diselesaikan terlebih dahulu. Sifat asosiatif yaitu :

• (a + b) + c = a + (b +c)

▶Contoh Penyelesaian :

• Operasi sifat komutatif (Pertukaran)

a) 10 + 12 = 12 + 10 = 22

b) 35 + 20 = 20 + 35 = 55

c) 100 + 123 = 123 + 100 = 223

d) 250 + 50 = 50 + 250 = 300

▶Contoh Penyelesaian :

•Operasi sifat asosiatif (Pengelompokan)

a. (20 + 50) + 10 = 20 + (50 + 10)

70 + 10 = 20 + 60

80 = 80

b. (15 + 10) + 23 = 15 + (10 + 23)

25 + 23 = 15 + 33

48 = 48

c. (30 + 20) + 35 = 30 + (20 + 35)

50 + 35 = 30 + 55

85 = 85

d. (100 + 200) + 300 = 100 + (200 + 300)

300 + 300 = 100 + 500

600 = 600

26. OK Google sifat komutatif penjumlahan


Sifat komutatif

Sifat komutatif adalah penjumlahan atau perkalian dua bilangan, dan kedua bilangan ditukarkan hasilnya akan tetap sama. Sifat Komutatif juga disebut dengan sifat pertukaran. Sifat Komutatif tidak berlaku untuk Pengurangan dan Pembagian karna hasilnya tidak sama.
Komutatif dapat di rumuskan Sebagai berikut :
a  +  b =  b  +  a
Dimana : a dan b bilangan bulat
(a  x  b  =  b  x  a)

Contoh

Contoh Sifat komutatif dalam Penjumlahan
2 + 3 = 5
dan kita tukar tempatnya seperti dibawah ini :
3 + 2 = 5 
Dan hasilnya tetap sama, yaitu 5. Maka Hukum Komutatif berlaku untuk operasi hitung penjumlahan.
Sehingga : 2 + 3 = 3 + 2

Contoh Sifat komutatif dalam Perkalian
2 x 3 = 6
dan kita tukar tempatnya seperti dibawah ini :
3 x 2 = 6 
Dan hasilnya tetap sama, yaitu 6. Maka Hukum Komutatif berlaku untuk operasi hitung Perkalian. Sehingga : 2 x 3 = 3 x 2

30. contoh sifat komutatif dalam perkalian yaitu​


Penjelasan dengan langkah-langkah:

Sifat komutatif, berarti sifat pertukaran, memiliki pemisalan:

a + b = b + aa × b = b × a

Yang berarti, sifat komutatif adalah sifat yang jika diubah tempat sukunya, hasilnya akan tetap sama.

_______

Contoh sifat komutatif pada perkalian:

7 × 8 = 8 × 7

56 = 56

Jawaban:

a + b = b + a

a x b = b x a

Contoh:

Penjumlahan:

25+18=43

18+25=43

Jadi...

25+18=18+25

43=43

Perkalian:

12x22=264

22x12=264

Jadi...

12x22=22x12

264=264

Penjelasan dengan langkah-langkah:

Pada penjumlahan dan perkalian bilangan bulat, pertukaran letak suku - sukunya tidak memengaruhi hasil akhirnya.
