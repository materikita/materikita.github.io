31. rumus suku ke-n dari barisan bilangan 100, 96, 92, 88,... adalah​


Materi : Barisan dan Geometri

Pola barisan aritmatika

100, 96, 92, 88, ...

Suku pertama = 100 , Beda = - 4

Maka rumus pola : Un = 104 - 4n

Cara Pengerjaannya :

Un = bn + ( a - b )

Un = (-4)n + ( 100 - [ -4 ] )

Un = - 4n + 100 + 4

Un = 104 - 4n
