semoga membantu

Subject : English

Category : Conditional Sentence

Level : SHS

Keywords : if


There  three types of conditional sentence

Conditional Type 1 => future possible

If + simple present, S + will + infinitive

Conditional Type 2 => unreal present

If + simple past, S + would + infinitive

Conditional Type 3 => unreal past

If + past perfect, S + would have + past participle

ANSWER :
