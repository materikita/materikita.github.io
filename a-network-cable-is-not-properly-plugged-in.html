26. what is the function main of HUB A. To close the network from one server to a computer client in one network B. To open the network from one server to a computer client in one network C. To divide the network from one server to a computer client in one network D. To the remove from one server to a computer client in one network E. To divide the client from one server to a computer in one network ​


Jawaban:

The main function of a hub is to connect multiple devices in a network. A hub is a networking device that receives incoming data from a network device and then broadcasts the data to all other connected devices. This allows all devices in the network to communicate with each other and access the same information. In this way, a hub acts as a central point of communication and helps to establish a network connection between devices. Therefore, the correct answer is B. To open the network from one server to a computer client in one network.
