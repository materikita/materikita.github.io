11. 3×^2 +11×-20=0tentukana.Bentuk kuadrat sempurnab.rumus persamaan kuadrad (rumus ABC)​


3x² + 11x - 20 = 0

Tentukan :

a) Bentuk Kuadrat Sempurna → Hp : { -5, 4/3 }

b) Rumus persamaan kuadrat (Rumus ABC) → Hp : { -5, 4/3 }

.

Pembahasan

Persamaan kuadrat adalah persamaan yang memiliki pangkat tertinggi 2.

Bentuk umum persamaan kuadrat

ax² + bx + c = 0 ; a ≠ 0

..

Cara menyelesaikan persamaan kuadrat

1) Memfaktorkan

ax² + bx + c = 0 diuraikan menjadi

(x - x₁)(x - x₂) = 0

2) Melengkapkan Kuadrat Sempurna

(x ± p)² = x² ± 2p + p²

3) Rumus ABC

[tex]x_{1,2} = \frac{ - b \: \pm \sqrt{ {b}^{2} - 4ac } }{2a} [/tex]

..

Dari Penjelasan di atas mari selesaikan soal berikut.

Diketahui :

3x² + 11x - 20 = 0

.

Ditanya :

Akar - akar persamaan kuadrat menggunakan cara :

a) Bentuk Kuadrat Sempurna

b) Rumus persamaan kuadrat (Rumus ABC)

.

Jawab :

a) Bentuk Kuadrat Sempurna

3x² + 11x - 20 = 0

3x² + 11x = 20

_____________ bagi kedua ruas dengan 3

[tex] {x}^{2} + \frac{11}{3} x = \frac{20}{3} [/tex]

[tex] {x}^{2} + 2.\frac{11}{6} x + {( \frac{11}{6} )}^{2} = \frac{20}{3} + {( \frac{11}{6}) }^{2} [/tex]

________________________

* ingat a² + 2ab + b² = (a + b)²

________________________

[tex](x + \frac{11}{6} ) {}^{2} = \frac{20}{3} + \frac{121}{36} [/tex]

[tex](x + \frac{11}{6} ) {}^{2} = \frac{240 + 121}{36} [/tex]

[tex](x + \frac{11}{6} ) {}^{2} = \frac{361}{36} [/tex]

[tex]x + \frac{11}{6} = \pm \sqrt{ \frac{361}{36} } [/tex]

[tex]x + \frac{11}{6} = \pm \frac{19}{6} [/tex]

[tex]x = - \frac{11}{6} \pm \frac{19}{6} [/tex]

[tex]x_{1} = - \frac{11}{6} + \frac{19}{6} = \frac{8}{6} = \frac{4}{3} [/tex]

[tex]x_{2} = - \frac{11}{6} - \frac{19}{6} = - \frac{ 30}{6} = - 5[/tex]

Jadi, Himpunan Penyelesaian dari Persamaan Kuadrat tersebut adalah Hp : { -5, 4/3 }

..

b) Rumus ABC

3x² + 11x - 20 = 0

a = 3

b = 11

c = -20

.

[tex]x_{1,2} = \frac{ - b \: \pm \sqrt{ {b}^{2} - 4ac } }{2a} [/tex]

[tex]x_{1,2} = \frac{ -11 \: \pm \sqrt{ {(11)}^{2} - 4(3)( - 20) } }{2(3)} [/tex]

[tex]x_{1,2} = \frac{ -11 \: \pm \sqrt{ 121 + 240 } }{6} [/tex]

[tex]x_{1,2} = \frac{ -11 \: \pm \sqrt{361 } }{6} [/tex]

[tex]x_{1,2} = \frac{ -11 \: \pm 19 }{6} [/tex]

[tex]x_{1} = \frac{ -11 \: + \: 19 }{6} = \frac{8}{6} = \frac{4}{3} [/tex]

[tex]x_{2} = \frac{ -11 \: - \: 19 }{6} = - \frac{30}{6} = - 5[/tex]

Jadi, Himpunan Penyelesaian dari Persamaan Kuadrat tersebut adalah Hp : { -5, 4/3 }
