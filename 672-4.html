35. berapa hasil dari 672 : (-56) =​


Jawaban:

= 672 : ( -56 )

= -12

( + ) : ( - ) = ( - )

Pembagian Bilangan Bulat

672 ÷ (-56) = -12

Mengapa?

Pada pembagian bilangan bulat jika kedua unsur/bilangan (+) itu dapat kita anggap hanya bilangan asli dan tidak adanya perubahan unsur dalam perhitungan/hasilnya PASTI (+).

Jika ada salah satu unsur/angka terdapat (-) maka hasil pembagian/perkalian PASTI akan menghasilkan (-) juga. Beda halnya dengan jika kedua unsur/angka itu (-) maka hasil pembagian/perkalian PASTI akan menghasilkan (+).

Untuk perhitungan perkalian/pembagian sebenarnya sama saja seperti biasanya, namun bedanya hanya pada unsur (-) atau (+).
