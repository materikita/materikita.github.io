7.500.000 mm³

= 7.500.000 x 1/1.000³ m³

= 7.500.000 x 1/(1.000 x 1.000 x 1.000) m³

= 7.500.000 x 1/1.000.000.000 m³

= 7.500.000/1.000.000.000 m³

= 75/10.000 m³

= 0,0075 m³
