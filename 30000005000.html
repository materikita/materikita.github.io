36. ubahlah ke dalam bentuk baku 2 3000000 sertakan caranya​


Bentuk biasa dari 23.000.000 adalah 2,3 x [tex]{10}^{7}[/tex]

Pendahuluan :

Sifat-sifat Perpangkatan :

[tex]1) \: {a}^{0} = 1 [/tex]

[tex]2) \: {a}^{m} \times {a}^{n} = {a}^{m + n} [/tex]

[tex]3) \: {a}^{m} \div {a}^{n} = {a}^{m - n} [/tex]

[tex]4) \: ( {a}^{m} ) ^{n} = {a}^{m \times n} [/tex]

[tex]5) \: ({a \times b})^{m} = {a}^{m} \times {b}^{m} [/tex]

[tex]6) \: ( {a \div b})^{m} = {a}^{m} \div {b}^{m} [/tex]

[tex]7) \: {a}^{m} = \frac{1}{ {a}^{ - m} } [/tex]

[tex]8) \: {a}^{ - m} = \frac{1}{ {a}^{m} } [/tex]

[tex]9) \: {a}^{ \frac{m}{n} } = \sqrt[n]{ {a}^{m} } [/tex]

Bentuk Baku :

[tex] a \times {10}^{n} , dengan 1 \leqslant a < 10 [/tex]

n = banyaknya angka nol untuk bilangan positif

n = banyaknya angka dibelakang koma untuk bilangan desimal

Bilangan 23.000.000

Ditanya :

Bentuk baku dari bilangan tersebut ?
