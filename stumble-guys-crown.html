Jawab:

Date Crown = 40.000

Almond  = 35.000

Penjelasan dengan langkah-langkah:

3x + 2y = 190.000  (x3)

5x + 3y = 310.000  (x2)

=   9x + 6y = 570.000

    10x + 6y = 610.000   -

    -x            = 40.000

    x = 40.000

y = 3x + 2y = 190.000

3 (40.000) + 2y = 190.000

120.000 + 2y = 190.000

2y + 190.000 - 120.000 = 70.000

70.000 / 2 = 35.000

y = 35.000
