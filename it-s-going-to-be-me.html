2. Positive : S + be +going to +V1 + object \ complement, negative : S + be + not + going to + V1 + object \ complement, interrogative be : + S + going to + V1 + object\ complement?, berikan contoh nya dan terjemahkan


I am going to Dufan next week. saya akan pergi ke dufan minggu depan

I am not going to Dufan next week. saya tidak akan pergi ke dufan minggu depan

Am I going to Dufan next week? apakah saya akan pergi ke dufan minggu depan?

s + to be + going to = artinya kegiatan sudah direncanakan

s + will/shall + v1= artinya kegiatan bersifat spontan atau belum direncanakan.

Positive sentences: She was going to work in post office.

Terjemahannya: Dia (perempuan) dulu pergi ke kantor pos untuk bekerja.

Negative sentences: He wasn't going to shop in hospital.

Terjemahannya: Dia (laki-laki) dulu tidak pergi ke rumah sakit untuk belanja.

Interrogative sentences: Was my brother going to eat in restaurant?

Terjemahannya: Apakah dulu saudara laki-lakiku pergi makan di restoran?

17. Contoh kalimat tenses be going to present (+)(-)(?) dan be going to past (+)(-)(?)...


*Present
pattern:
(+) = S + V1(s/es) + O(Adj./Adv/etc.)
(-) = S + Do/Does + not + V1 + O
(?) = Do/Does + S + V1 + O ?
Example :
(+) = He buys a car now
(-) = He does not buy a car now
(?) = Does he buy a car now ?

Note: keterangan waktu tidak harus ditulis.

*Past
pattern:
(+) = S + V2 + O
(-) = S + Did + not + V1 + O
(?) = Did + S + V1 + O ?
Example:
(+) = She went to school yesterday
(-) = She did not go to school yesterday
(?) = Did she go to school yesterday ?

Note: keterangan waktu tidak harus ditulis

34. Rumus to be going to


Kalimat positif :
Subyek + to be + going to + verb1
contoh :
I’m going to go swimming tomorrow.

Kalimat negatif
Subyek + to be + not + going to + verb1
Contoh :
He is not going to play football.

Kalimat interrogatif
To be + subyek + going to + verb1?
