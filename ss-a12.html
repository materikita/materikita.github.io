33. tentukan biloks s dalam A12(SO2)3​


Jawaban:

S = +2

Penjelasan:

Al2(SO2)3​ terdiri atas 2 Al, 3 S dan 6 O. Biloks O menurut aturan biloks adalah -2, maka keseluruhan biloks O adalah -12. Dan biloks Al adalah +3

Jumlah seluruh biloks Al2(SO2)3 adalah 0, secara matematis ditulis sebagai berikut

2 . biloks Al + 3 . biloks S + 6 . biloks O = 0

2 . 3 + 3 . biloks S + 6 . (-2) = 0

3 . biloks S = 12 - 6

biloks S = 6/3 = 2

40. Diketahui barisan aritmatika a1 + a2 + a3 + .... + a12 = 3 dan a5 + a7 = 1, nilai a12 adalah


Jika diketahui barisan aritmatika a1 + a2 + a3 + .... + a12 = 3 dan a5 + a7 = 1, nilai a12 adalah [tex]-\frac{5}{2}[/tex].

Penjelasan dengan langkah-langkah:

Rumus suku ke-n pada barisan aritmatika:

[tex]Un=a+(n-1)b[/tex]

Rumus jumlah n suku pada barisan aritmatika:

[tex]Sn=\frac{n}{2} (2a+(n-1)b)\\Sn=\frac{n}{2}(a+Un)[/tex]

Dimana;

a = suku pertama

b = beda/selisih suku yang berdekatan

n = suku ke-

Dari soal diketahui;

a = a1

a5+a7 = U₅+U₇ = 1

S₁₂=3

Perhatikan pola barisan tersebut, nilai U₅ = a5 dan U₇ = a7.

Untuk U₅ = a5

[tex]Un=a+(n-1)b\\U5=a+(5-1)b\\a5=a+4b[/tex]

Untuk U₇ = a7

[tex]Un=a+(n-1)b\\U7=a+(7-1)b\\a7=a+6b[/tex]

Sehingga,

[tex]a5+a7=1\\(a+4b)+(a+6b)=1\\2a+10b=1[/tex]

Jumlah 12 suku = 3

[tex]Sn=\frac{n}{2}(2a+(n-1)b)\\S12=\frac{12}{2}(2a+(12-1)b)\\3=6(2a+11b)\\1=2(2a+11b)\\4a+22b=1[/tex]

Eliminasi kedua persamaan diatas:

[tex](2a1+10b=1).2\\4a1+20b=2\\4a1+22b=1[/tex]

__________ -

[tex]-2b=1\\b=-\frac{1}{2}[/tex]

Subsitusikan nilai b ke persamaan pertama.

[tex]2a+10b=1\\2a+(10.-\frac{1}{2} )=1\\2a-5=1\\2a=1+5\\2a=6\\a=3[/tex]

Karena a = 3 dan b = [tex]-\frac{1}{2}[/tex], Maka;

[tex]Un=a+(n-1)b\\U12 = 3+((12-1).-\frac{1}{2} )\\U12=3+(11.-\frac{1}{2})\\U12=3-\frac{11}{2}\\U12=\frac{6-11}{2}\\U12=-\frac{5}{2}[/tex]

Jadi, nilai U₁₂ = a12 = [tex]-\frac{5}{2}[/tex].
