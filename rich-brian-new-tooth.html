23. diketahuai uang akbar : uang brian : 5:4 dan uang brian : uang catur =3:4.jika jumlah uang akbar ,brian ,dan catur Rp 4300000, maka uang brian adalah..​


Perbandingan uang akbar dan uang brian adalah 5 : 4. Perbandingan uang brian dan uang catur adalah 3 : 4. Jika jumlah uang akbar, brian dan catur Rp. 4.300.000. Maka uang brian adalah sebesar Rp. 1.200.000.

Penyelesaian Soal :

Misalkan Uang akbar = x

               Uang Brian = y

               Uang catur = z

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Perbandingan uang akbar dan uang brian adalah 5 : 4" yaitu :

[tex]\frac{x}{y}[/tex] = [tex]\frac{5}{4}[/tex]

5y = 4x

x = [tex]\frac{5y}{4}[/tex]   ...(Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Perbandingan uang brian dan uang catur adalah 3 : 4" yaitu :

[tex]\frac{y}{z}[/tex] = [tex]\frac{3}{4}[/tex]

3z = 4y

z = [tex]\frac{4y}{3}[/tex]   ... (Persamaan 2)

LANGKAH KETIGA (III)

Buatlah persamaan dari pernyataan "Jika jumlah uang akbar, brian dan catur Rp. 4.300.000" yaitu :

x + y + z = 4.300.000   ... (Persamaan 3)

LANGKAH KEEMPAT (IV)

Subtitusikan nilai x dan z pada persamaan 3 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

x + y + z = 4.300.000

[tex]\frac{5y}{4}[/tex] + y + [tex]\frac{4y}{3}[/tex] = 4.300.000

[tex]\frac{43y}{12}[/tex] = 4.300.000

43y = 4.300.000 × 12

43y = 51.600.000

y = [tex]\frac{51.600.000}{43}[/tex]

y = 1.200.000

∴ Kesimpulan Uang brian adalah sebesar Rp. 1.200.000.
