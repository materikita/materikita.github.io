Penjelasan:

V = 6v

i mula-mula= 0,5 A

kawat dipotong menjadi 2 bagian disusun secara paralel.

i = ?

Jawab :

R mula-mula = V/i = 6/ 0,5 = 12 ohm

29. hei bantuin ya terimakasih..​


Penjelasan dengan langkah-langkah:

• cos (5/4 x + 80°) = cos 260°

5/4 x + 80° = 260° + k.360°

5/4 x = 180° + k.360°

x = 144° + k.288°

k = 0 ➡ x = 144° + 0° = 144°

k = 1 ➡ x = 144° + 288° = 432°

• cos (5/4 x + 80°) = cos (360 - 260)°

cos (5/4 x + 80°) = cos 100°

5/4 x + 80° = 100° + k.360°

5/4 x = 20° + k.360°

x = 16° + k.288°

k = 0 ➡ x = 16° + 0° = 16°

k = 1 ➡ x = 16° + 288° = 304°

untuk 0° ≤ x ≤ 360°

Jadi, HP = {16°, 144°, 304°}
