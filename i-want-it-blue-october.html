39. [tex]\mathcal\red{Q}\mathfrak\orange{u}\pmb\green{i}\it\blue{z}[/tex]7³ + 3³ + 45²____________[tex] \blue{ \rule{10pt}{20pt}}[/tex]​


Penjelasan dengan langkah-langkah:

Soal Dan Penyelesaian Nya :

[tex] = {7}^{3} + {3}^{3} + {45}^{2} [/tex]

[tex] = 7 \times 7 \times 7 + {3}^{3} + {45}^{2} [/tex]

[tex] = 49 \times 7 + {3}^{3} + {45}^{2} [/tex]

[tex] = 343 + {3}^{3} + {45}^{2} [/tex]

[tex] = 343 + 3 \times 3 \times 3 + {45}^{2} [/tex]

[tex] = 343 + 9 \times 3 + {45}^{2} [/tex]

[tex] = 343 + 27 + {45}^{2} [/tex]

[tex] = 370 + {45}^{2} [/tex]

[tex] = 370 + 45 \times 45[/tex]

[tex] = 370 + 2.025[/tex]

[tex] = 2.395[/tex]
