7. Panjang busur kurva y = x√x, untuk x pada interval 0 ≤ x ≤ 5 adalah . . ._________iya" aku tau km nyesel dpt tmbol jwb, skip aj gpp ☺️​


Panjang busur kurva [tex]y=x\sqrt{x}[/tex] pada interval 0 ≤ x ≤ 5 adalah [tex]\boldsymbol{\frac{335}{27}}[/tex] satuan panjang.

PEMBAHASAN

Integral merupakan operasi yang menjadi kebalikan dari operasi turunan/diferensial. Sehingga integral sering juga disebut sebagai antiturunan.

[tex]\displaystyle{f(x)=\int\limits {\left [ \frac{df(x)}{dx} \right ]} \, dx}[/tex]

Salah satu fungsi dari integral adalah untuk menghitung panjang busur kurva dari titik a ke titik b, dimana panjang busur S dapat dicari dengan rumus :

[tex]\displaystyle{S=\int\limits^{x_2}_{x_1} {\sqrt{1+[f'(x)]^2}} \, dx },~atau[/tex]

[tex]\displaystyle{S=\int\limits^{y_2}_{y_1} {\sqrt{1+[f'(y)]^2}} \, dy }[/tex]

Dengan f'(x) dan f'(y) adalah turunan pertama fungsi.

.

DIKETAHUI

Kurva [tex]y=x\sqrt{x}[/tex]

.

DITANYA

Tentukan panjang busur kurva pada interval 0 ≤ x ≤ 5.

.

PENYELESAIAN

Karena batas integralnya dalam x, kita integralkan terhadap variabel x.

[tex]y=x\sqrt{x}[/tex]

[tex]y=x^{\frac{3}{2} }[/tex]

[tex]\frac{dy}{dx} =\frac{3}{2}x^{\frac{1}{2} }[/tex]

.

Maka panjang busurnya :

[tex]\displaystyle{S=\int\limits^{x_2}_{x_1} {\sqrt{1+[f'(x)]^2}} \, dx }[/tex]

[tex]\displaystyle{S=\int\limits^{5}_{0} {\sqrt{1+\left ( \frac{3}{2}x^{\frac{1}{2}} \right )^2}} \, dx }[/tex]

[tex]\displaystyle{S=\int\limits^5_0 {\sqrt{1+\frac{9}{4}x }} \, dx }[/tex]

[tex]-----------------[/tex]

Misal :

[tex]\displaystyle{u=1+\frac{9}{4}x}[/tex]

[tex]\displaystyle{du=\frac{9}{4}dx}[/tex]

[tex]\displaystyle{dx=\frac{4}{9}du}[/tex]

[tex]-----------------[/tex]

[tex]\displaystyle{S=\int\limits^5_0 {\sqrt{u }} \, \left ( \frac{4}{9}du \right ) }[/tex]

[tex]\displaystyle{S=\frac{4}{9} \int\limits^5_0 {u^{\frac{1}{2}}} \, du }[/tex]

[tex]\displaystyle{S=\frac{4}{9}\times\frac{1}{\frac{1}{2}+1}u^{\frac{1}{2}+1}\Bigr|^5_0 }[/tex]

[tex]\displaystyle{S=\frac{8}{27}u^{\frac{3}{2}}\Bigr|^5_0 }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left ( 1+\frac{9}{4}x \right )^{\frac{3}{2}}\Bigr|^5_0 }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left [ \left ( 1+\frac{9}{4}(5) \right )^{\frac{3}{2}} -\left ( 1+\frac{9}{4}(0) \right )^{\frac{3}{2}}\right ] }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left [ \left ( \frac{49}{4} \right )^{\frac{3}{2}} -1\right ] }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left ( \frac{49}{4}\sqrt{\frac{49}{4}}-1 \right ) }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left ( \frac{49}{4}\times\frac{7}{2}-1 \right ) }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left ( \frac{343}{8}-\frac{8}{8} \right ) }[/tex]

[tex]\displaystyle{S=\frac{8}{27}\left ( \frac{335}{8} \right ) }[/tex]

[tex]\displaystyle{S=\frac{335}{27} }[/tex]

.

KESIMPULAN

Panjang busur kurva [tex]y=x\sqrt{x}[/tex] pada interval 0 ≤ x ≤ 5 adalah [tex]\boldsymbol{\frac{335}{27}}[/tex] satuan panjang.

.

.
