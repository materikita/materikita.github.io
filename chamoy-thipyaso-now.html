14. Apa perbedaan cara mengucapkan know dengan now


- Know = dibaca ( nou )
- Now = dibaca tetap ( naw / nau )
Artinya :
- Know = Tahu ( tau )
- Now = Sekarang
Misal :
- I know = baca ( Ai nou )
- Just now = baca ( Jast naw / nau )-Jika "Know" cara membacanya adalah (Nou) artinya tahu;mengenal;mengetahui.
-Jika "Now" cara membacanya adalah (Nau) artinya sekarang
