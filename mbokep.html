32. Golekana tegese cangkriman pepindhan ing ngisor iki :1. Didhelok mendhuwur tibane ing ngisor2. Bosok malah enak3. Mboke dielus – elus anake diidak – idak4. Emboke wuda anake tapihan5. Dilebokake tambah metu6. Dikethok malah luwih dhuwur7. Wite andhikih wohe andhaka​


Golekana tegese cangkriman pepindhan ing ngisor iki !

Jawaban

1. Didhelok mendhuwur tibane ing ngisor

Artinya : Dilihat ke atas jatuhnya kebawah.

Batangane >> Gendheng trocoh (Genteng bocor) utawa wong ngentut.

2. Bosok malah enak

Artinya : Busuk malah enak.

Batanganne >> Tempe utawa tape.

3. Mboke dielus – elus anake diidak – idak

Artinya : Ibunya di elus-elus anaknya diinjak-injak.

Batanganne >> Andha (tangga).

4. Emboke wuda anake tapihan

Artinya : Ibunya terbuka anake tertutup.

Batanganne >> Wit pring.

5. Dilebokake tambah metu

Artinya : Dimasukkan tambah keluar.

Batanganne >> Benik (Kancing).

6. Dikethok malah luwih dhuwur

Artinya : Dipotong malah bertambah tinggi.

Batanganne >> Katok/Clana dawa (Celana panjang).

7. Wite andhikih wohe andhaka

Artinya : Pohonnya kecil buahnya besar, Pohonnya besar buahnya kecil.

Batanganne >> Woh semangka utawa woh beringin.

Pembahasan

Cangkriman yaiku unen unen kang isine kudu di bathang utawa dibedhek lan dijawab.

Cangkriman Pepindhan yaiku Cangkriman kang ngemu teges pepindhan utawa irib iriban. (Pengandaian).
