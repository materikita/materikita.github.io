6. suatu larutan terdiri atas campuran 1,28 mol c7h16 2,92 mol c8h18 dan 2,64 mol c9h20 tentukan fraksi mol masing masing komponen dalam larutan!​


Masing - masing fraksi mol dari campuran 1,28 mol C₇H₁₆ 2,92 mol C₈H₁₈ dan 2,64 mol C₉H₂₀ secara berturut-turut adalah Xa = 0,2, Xb = 0,4, dan Xc = 0,4

Penjelasan:

Diketahui: n C₇H₁₆ = 1,28 mol

                 n  C₈H₁₈ = 2,92 mol

                 n C₉H₂₀ = 2,64 mol

Ditanya; Fraksi mol masing - masing komponen adalah?

Jawab:

1. Fraksi mol C₇H₁₆

Xa = n C₇H₁₆ / n C₇H₁₆ + n  C₈H₁₈ +  n C₉H₂₀

Xa = 1,28 / 1,28 + 2,92 + 2,64

Xa = 1,28/6,84

Xa = 0,2

2. Fraksi mol C₈H₁₈

Xb = n C₈H₁₈ / n C₇H₁₆ + n  C₈H₁₈ +  n C₉H₂₀

Xb = 2,92 / 1,28 + 2,92 + 2,64

Xb = 2,92 / 6,84

Xb = 0,4

3. Fraksi mol C₉H₂₀

Xc = n C₉H₂₀ / n C₇H₁₆ + n  C₈H₁₈ +  n C₉H₂₀

Xc = 2,64 / 1,28 + 2,92 + 2,64

Xc = 2,64 / 6,84

Xc = 0,4

Sehingga total dari masing - masing fraksi mol tersbut adalah Xa + Xb + Xc = 0,2 + 0,4 + 0,4 = 1
