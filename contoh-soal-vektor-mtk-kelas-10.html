13. !!LUMAYAN POIN!!kakak2 dan teman2tolong jawab 2 soal mtk kelas 10 tentang vektor​


1. Diketahui a = (-8, 13) dan b = (12, 7). Hasil dari a + b adalah (4, 20).

Jawaban: D

2. Diketahui p = -12i + 37j dan q = 18i - 57j. Hasil dari p + q adalah -103i - 20j.

Jawaban: D

Pembahasan

Jika diketahui dua buah vektor pada dimensi 2 dinyatakan dalam vektor arah a = (x₁, y₁) dan vektor b = (x₂, y₂), maka operasi penjumlahan dan pengurangan vektor a dan vektor b dapat dilakukan sebagai berikut.

a + b = (x₁, y₁) + (x₂, y₂) = ((x₁ + x₂), (y₁ + y₂))a - b = (x₁, y₁) - (x₂, y₂) = ((x₁ - x₂), (y₁ - y₂))

Atau jika vektor dinyatakan dalam bentuk kombinasi linear, yaitu vektor arah a = x₁i + y₁j dan vektor b = x₂i + y₂j. Operasi penjumlahan dan pengurangan vektor a dan vektor b dapat dilakukan sebagai berikut.

a + b = (x₁ + x₂)i + (y₁ + y₂)ja - b = (x₁ - x₂)i - (y₁ - y₂)j

↓↓↓No. 1

Diketahui:

a = (-8, 13)

b = (12, 7)

Ditanyakan:

a + b = ...

Penyelesaian:

a + b = (-8, 13) + (12, 7)

a + b = (((-8) + 12), (13 + 7))

a + b = (4, 20)

Jadi hasil dari a + b adalah (4, 20).

No. 2

Diketahui:

p = -121i + 37j

q = 18i - 57j

Ditanyakan:

p + q = ...

Penyelesaian:

p + q = ((-121) + 18)i + (37 + (-57))j

p + q = -103i + (-20)j

p + q = -103i - 20j

Jadi hasil dari p + q adalah -103i - 20j.

2).Diberikan tiga buah vektor masing-masing sebagai berikut:

p = i + j + k

q = 2i – 3j +k

r = – 2j – 3k

Jika s = 2p – q + 3r, maka s =….

A. i + 2j -7k

B. – i + 6k

C. 6i + 7k

D. – j – 8k

E. j + 4k

Pembahasan:

p = i + j + k

q = 2i – 3j +k

r = – 2j – 3k

Menentukan vektor s, harus teliti dengan tanda positif dan negatif dalam soal.

s = 2p – q + 3r

= 2(i + j + k) – (2i – 3j +k) + 3(– 2j – 3k)

= 2i + 2j + 2k – 2i + 3j – k – 6j – 9k

= 2i – 2i + 2j + 3j – 6j + 2k – k – 9k

s = – j – 8k

26. berserta caranya ya,vektor peminatan mtk kelas 10 ​


Diketahui A(-2,-1,0) , B(x-2, 2y-3 , z-3), dan 3PB + 2AP = 0. Jika P(5,2,8) maka x + y + z = 19/2 = 9,5

PEMBAHASAN

Vektor adalah besaran yang memiliki besar dan arah.

Untuk menggambarkan vektor di dalam koordinat kartesius adalah dengan menggambarkan garis berarah dimana kemiringan garis menunjukkan arah vektor dan panjang garis menunjukkan besar vektor.

Vektor posisi sebuah titik adalah vektor yang ditarik dari titik pangkal koordinat O(0,0) ke titik yang dimaksud.

Penambahan dua buah vektor dapat dilakukan dengan cara sebagai berikut :

[tex]\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC}[/tex]

Prinsipnya lihat huruf belakang vektor pertama dengan huruf awal vektor kedua jika sama maka hasilnya adalah vektor dengan huruf awal vektor pertama dengan huruf akhir vektor kedua.

[tex]\overrightarrow{AB} = - \overrightarrow{BA}[/tex]

Prinsipnya jika vektor negatif maka hurufnya dibalik saja karena vektor negatif adalah vektor dengan besar sama tetapi arahnya berlawanan.

Okay marilah kita gunakan prinsip ini untuk menyelesaikan persoalannya sebagai berikut.

[tex]\overrightarrow{OB}=\left[\begin{array}{ccc}x-2\\2y-3\\z-3\end{array}\right][/tex]

[tex]\overrightarrow{OP}=\left[\begin{array}{ccc}5\\2\\8\end{array}\right][/tex]

[tex]\overrightarrow{OA}=\left[\begin{array}{ccc}-2\\-1\\0\end{array}\right][/tex]

[tex]\overrightarrow{PB}=\overrightarrow{OB}-\overrightarrow{OP}[/tex]

[tex]\overrightarrow{PB}=\left[\begin{array}{ccc}x-2\\2y-3\\z-3\end{array}\right]-\left[\begin{array}{ccc}5\\2\\8\end{array}\right][/tex]

[tex]\overrightarrow{PB}=\left[\begin{array}{ccc}x-2-5\\2y-3-2\\z-3-8\end{array}\right][/tex]

[tex]\overrightarrow{PB}=\left[\begin{array}{ccc}x-7\\2y-5\\z-11\end{array}\right][/tex]

[tex]\overrightarrow{AP}=\overrightarrow{OP}-\overrightarrow{OA}[/tex]

[tex]\overrightarrow{AP}=\left[\begin{array}{ccc}5\\2\\8\end{array}\right]-\left[\begin{array}{ccc}-2\\-1\\0\end{array}\right][/tex]

[tex]\overrightarrow{AP}=\left[\begin{array}{ccc}5-(-2)\\2-(-1)\\8-0\end{array}\right][/tex]

[tex]\overrightarrow{AP}=\left[\begin{array}{ccc}7\\3\\8\end{array}\right][/tex]

[tex]3\overrightarrow{PB}+2\overrightarrow{AP}=0[/tex]

[tex]3\left[\begin{array}{ccc}x-7\\2y-5\\z-11\end{array}\right]+2\left[\begin{array}{ccc}7\\3\\8\end{array}\right]=0[/tex]

[tex]\left[\begin{array}{ccc}3x-21\\6y-15\\3z-33\end{array}\right]+\left[\begin{array}{ccc}14\\6\\16\end{array}\right]=0[/tex]

[tex]\left[\begin{array}{ccc}3x-21+14\\6y-15+6\\3z-33+16\end{array}\right]=0[/tex]

[tex]\left[\begin{array}{ccc}3x-7\\6y-9\\3z-17\end{array}\right]=0[/tex]

3x - 7 = 0
