Perintah-perintah yang termasuk Office Button

1. New = Perintah yang digunakan untuk membukan dokumen baru.

2. Open = Perintah yang digunakan untuk membuka file dokumen.

3. Save = Perintah yang digunakan untuk menyimpan file tanpa format tertentu.

4. Save as = Perintah yang digunakan untuk menyimpan file dengan format tertentu.

5. Print = Perintah yang digunakan untuk mencetak file dokumen.

6. Prepare = Perintah yang digunakan untuk melakukan property khusus terhadap dokumen.

7. Send = Perintah yang digunakan untuk mengirimkan file dokumen aktif sebagai e – mail atau facsimile.

8. Publish = Perintah yang digunakan untuk mempublish / menerbitkan dokumen ke internet.

9. Close = Perintah yang digunakan untuk menutup file dokumen aktif


25. How to analyze data more than 1 million rows in excel ?


Analyze data more than 1 million rows we can use Power Pivots fiture in microsoft excel that can handle the large amount data. So the function of microsoft excel is so useful.

Pembahasan:

Microsoft telah menambahkan Power Pivots ke Excel untuk membantu menganalisis data dalam jumlah besar. Power Pivot dapat menangani ratusan juta baris data, menjadikannya alternatif yang lebih baik untuk Microsoft Access, yang merupakan satu-satunya cara untuk melakukannya sebelum Excel. Pikirkan Power Pivot sebagai cara untuk menggunakan PivotTable dengan kumpulan data yang sangat besar. Ini juga berguna jika datanya berasal dari berbagai sumber. Power Pivot memungkinkan Anda mengimpor data ini ke dalam satu buku kerja tanpa memerlukan beberapa lembar sumber. Ini bisa membingungkan dan membuat frustrasi.

28. ikon save redo dan undo merupakan bagian menu dari


Ikon save, redo dan undo merupakan bagian menu dari Quick Access Toolbar yang terletak pada Microsoft Office. Walau terletak pada toolbar, quick access lebih mempermudah penggunanya untuk melakukan perintah tertentu

Pembahasan

Quick access toolbar adalah bagian toolbar yang berisi ikon untuk melakukan suatu perintah pada microsoft office yang fungsinya sebagai tombol akses cepat untuk penggunanya. Kita dapat menemukan quick access toolbar berada di pojok kiri atas pada tampilan Microsoft Office.

Quick access toolbar sendiri dapat di customize untuk isi ikonnya dengan memilih menu yang ada pada ikon panah disampingnya.

Ikon - ikon yang dapat dimasukkan dalam quick access toolbar

SaveUndoRedoPrint previewNew OpenQuick printDraw tableSpell and grammar checking
