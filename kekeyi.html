38. Diketahui persamaan kuadrat 2x² + 3x - 8=0 memiliki akar akar p dan q. tentukan persamaan kuadrat yang memiliki akar.a. -2p dan -2qb. p-3 dan q-3help me my darling -kekeyi​


Penjelasan:

semoga membantu terimakasih

2x² + 3x - 8 = 0

p + q = -3/2

p.q = -8/2 = -4

a. (-2p) dan (-2q)

(-2p) + (-2q) = -2(p + q)

= -2(-3/2)

= 3

(-2p)(-2q) = 4pq

= 4(-4)

= -16

x² + ((-2p) + (-2q))x + ((-2p)(-2q)) = 0

x² - 3x - 16 = 0

b. (p - 3) dan (q - 3)

(p - 3) + (q - 3) = p + q - 6

= -3/2 - 6

= -3/2 - 12/2

= -15/2

(p - 3)(q - 3) = pq - 3p - 3q + 9

= pq - 3(p + q) + 9

= -4 - 3(-3/2) + 9

= 5 + 9/2

= 10/2 + 9/2

= 19/2

x² - ((p - 3) + (q - 3))x + ((p - 3)(q - 3)) = 0

x² + 15/2 x + 19/2 = 0

2x² + 15x + 19 = 0
