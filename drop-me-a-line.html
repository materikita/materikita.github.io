22. Berikut ini merupakan struktur dari basis data adalah A. Create database, drop database, create table, drop table, insert, retrieve, update, delete B. Create database, drop database, create table, drop table, insert, select, update, delete · C. Create database, drop database, create table, drop table, insert, rename, update, delete D. Create database, drop database, create table, drop table, insert, update, delete E. Create database, drop database, create table, drop table, insert, update, delete ​


Jawaban:

B. Create database, drop database, create table, drop table, insert, select, update, delete.

Penjelasannya:

Struktur dari basis data adalah Create database, drop database, create table, drop table, insert, select, update, delete. Struktur ini merupakan struktur dasar dari basis data yang digunakan untuk mengelola data dalam sebuah sistem basis data. Create database dan drop database digunakan untuk membuat dan menghapus database. Create table dan drop table digunakan untuk membuat dan menghapus tabel dalam database. Insert digunakan untuk memasukkan data ke dalam tabel. Select digunakan untuk mengambil data dari tabel. Update digunakan untuk memperbarui data yang sudah ada dalam tabel. Delete digunakan untuk menghapus data dari tabel. Oleh karena itu, jawaban yang tepat untuk pertanyaan ini adalah B. Create databese, drop database, create table, drop table, insert, select, update, delete.

26. tuliskan langkah-langkah untuk membuat drop cap pada aplikasi Me. word 2010


Cara Membuat Drop Cap


Pilih huruf pertama dari paragraf.


Pada Insert tab, Text group, klik Drop Cap.


Pilih posisi Drop Cap:


Dropped: Drop Cap ditempatkan dalam paragraf.


In margin: Drop Cap ditempatkan dalam margin.


Ulangi langkah-langkah di atas bila ingin mengganti posisi Drop Cap.

34. jelaskan fungsi perintah line to drop pada menu dropcat


Fungsi Drop cap adalah agar karakter pertama pada awal paragraf tampil lebih besar. Drop cap sering kita lihat pada novel atau buku.

Bagi Anda yang sudah tidak sabar untuk mencobanya, ikuti langkah-langkah berikut:


1. Tempat kursor dalam paragraf yang akan Anda jadikan drop cap.

2. Klik tab Insert, pada bagian Text pilih Drop Cap.

￼


3. Selanjutnya pilih :

None, untuk membatalkan/menghapus drop cap yang pernah dibuat.Dropped, menempatkan drop cap rata dengan margin kiri.In margin, menempatkan drop cap di dalam margin kiri.

￼



4. Klik Drop Cap Options… untuk melakukan pengaturan lebih lanjut.


￼


Pada tombol daftar Font, pilih jenis huruf yang Anda inginkan.Pada Lines to drop, tentukan jumlah baris untuk menempatkan drop cap tersebut.

￼


Pada Distance from text, tentukan jarak Drop cap dengan teks/paragraf.

￼



5. Selanjutnya Klik OK.
