14. Terdapat dua orang mahasiswa yaitu dodi dan andre. jika peluang dodi lulus adalah sebesar 0,85 dan andre lulus adalah sebesar 0,6. maka peluang dodi lulus sedangkan andre tidak lulus adalah ……a. 0,25b. 3,25c. 0,34d. 0,9e. 0,15​


Apabila dodi lulus sedangkan andre tidak lulus, maka peluang dari kejadian tersebut adalah (c) 0,34

Penjelasan dengan langkah-langkah:

Peluang pada contoh soal tersebut menggunakan distribusi binomial. Distribusi binomial adalah sebuah peluang yang hanya memiliki dua hasil, yaitu berhasil atau gagal. Jumlah total semua kemungkinan yaitu yang berhasil ditambah yang gagal adalah sama dengan 1. dapat ditulis dengan :

Peluang berhasil + peluang gagal = 1.

Diketahui :

Peluang doni lulus = 0,85

Peluang andre lulus = 0,6

maka peluang andre untuk tidak lulus adalah 1 - 0,6 = 0,4

Ditanyakan :

peluang doni lulus sedangkan andre tidak lulus?

Jawab :

total peluang = peluang doni lulus x peluang andre tidak lulus

total peluang = 0,85 x 0,4

total peluang = 0,34

Jawab:

Jumlah uang Andre dan Bagas adalah Rp. 172.000,00

Penjelasan dengan langkah-langkah:

Kita buat model matematisnya dulu ya.

Diketahui:

A = Uang Andre

B = Uang Bagas

C = Uang Clara

1. A = B + 2C + 40000

(uang Andre Rp40.000,00 lebih banyak dari uang Bagas ditambah dua kali uang Clara)

2. A + B + C = 200000

(jumlah uang Andre Bagas dan Clara Rp Rp200.000,00)

3. B - C = 10000

(selisih uang Bagas dan Clara Rp10.000,00)

Ditanyakan berapa A + B?

(Berapakah jumlah uang Andre dan Bagas)

Jawab:

Dari 1 dan 2:

B + 2C + 40000 + B + C = 200000

2B + 3C = 160000

Sedangkan dari (3):

B = C + 10000

2 × (C + 10000) + 3C = 160000

2C + 3C = 160000 - 20000

5C = 140000

C = 28000

B = 28000 + 10000

B = 38000

A = 38000 + 56000 + 40000

A = 134000

Jadi A + B = 134000 + 38000 = 172000

Jumlah uang Andre dan Bagas adalah Rp. 172.000,00

35. Andre berangkat pukul 07. 20, Andre harus tiba pukul 08.00 Andre mengendarai kecepatan 50km/jam Jika Andre ingin sampai di tempat tujuan 07.45. Berapa kecepatan yang diperlukan?​


Jawaban:

Diketahui : Kecepatan (v) = 60 km/jam

Jam berangkat = 06.30

Jam tiba = 07.20

Ditanya : Jarak tempuh (s) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung waktu tempuh (t) ayah dengan menggunakan cara sebagai berikut :

Waktu tempuh (t) = jam tiba - Jam berangkat

= 07.20 - 06.30

= 00.50 = 50 menit

LANGKAH KEDUA (II)

Konversikan satuan waktu ke dalam bentuk jam dengan menggunakan cara sebagai berikut :

1 menit = ¹/₆₀ jam → 50 menit = 50 × ¹/₆₀ jam = ⁵/₆ jam

LANGKAH KETIGA (III)

Hitung jarak tempuh (s) ayah ke tempat kerja dengan menggunakan rumus sebagai berikut :

V = s/t

keterangan : v = kecepatan (km/jam)

s = jarak yang ditempuh (km)

t = waktu tempuh (jam)

Maka cara menghitungnya yaitu :

v = s/t

60 km/jam = s / ⁵/₆ jam

s = 60 km/jam × ⁵/₆ jam

s = 50 km

∴ Kesimpulan jarak rumah ke tempat kerja adalah 50 km.

2. Diketahui : Kecepatan (v) = 48 km/jam

Jam berangkat = 07.45

Jam tiba = 08.05

Ditanya : Jarak tempuh (s) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung waktu tempuh (t) paman dengan menggunakan cara sebagai berikut :

Waktu tempuh (t) = jam tiba - Jam berangkat

= 08.05 - 07.45

= 00.20 = 20 menit

LANGKAH KEDUA (II)

Konversikan satuan waktu ke dalam bentuk jam dengan menggunakan cara sebagai berikut :

1 menit = ¹/₆₀ jam → 20 menit = 20 × ¹/₆₀ jam = ¹/₃ jam

LANGKAH KETIGA (III)

Hitung jarak tempuh (s) ayah ke tempat kerja dengan menggunakan rumus sebagai berikut :

V = s/t

keterangan : v = kecepatan (km/jam)

s = jarak yang ditempuh (km)

t = waktu tempuh (jam)

Maka cara menghitungnya yaitu :

v = s/t

48 km/jam = s / ¹/₃ jam

s = 48 km/jam × ¹/₃ jam

s = 16 km

∴ Kesimpulan jarak rumah ke tempat kerja adalah 16 km.

Pembahasan :

Kecepatan adalah kemampuan pergerakan suatu benda. Pergerakan benda berpengaruh terhadap kecepatannya. Semakin cepat benda bergerak maka akan semakin tinggi kecepatannya.

Jarak adalah angka yang menunjukkan seberapa jauh suatu benda berubah posisi melalui lintasan tertentu. Umumnya jarak menggunakan satuan meter atau kilometer.

Waktu tempuh adalah lamanya waktu yang digunakan untuk menempuh suatu jarak tertentu. Satuannya adalah detik, menit, jam.
