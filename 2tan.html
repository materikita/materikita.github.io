12. jika 2tan^2x + 3tanx - 2 = 0, maka sinx+cosx?


2tan²x + 3tanx - 2 = 0
(2tanx - 1)(tanx + 2)
tanx = 1/2      v     tanx = -2

(a)
bila tanx = 1/2
tanx = sinx / cosx
1/2 = sinx / cosx
maka sinx = 1 dan cosx = 2

sinx + cosx = 1 + 2 = 3

(b)
bila tanx = -2
tanx = sinx / cosx
-2 = sinx / cosx
maka sinx = -2 dan cosx = 1

sinx + cosx = -2 + 1 = -1
